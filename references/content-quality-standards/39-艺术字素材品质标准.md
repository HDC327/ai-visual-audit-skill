# 艺术字素材品质标准
### 设计原则及标准

| 设计原则 | 标准说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 辨识度高 | 确保字体设计不会牺牲可读性，文字能够被快速准确地识别 |  |  |
| 视觉平衡 | 保持**视觉平衡**，无论是对称还是非对称，都应给人**稳定和有序**的感觉 可以通过调整字体结构、字面大小、笔画造型、文字间距、装饰元素等细节达到重心平稳 |  |  |
| 和谐统一 | 字体和字体，字体和元素，元素和元素间需**遵循相同的设计规范和风格，以确保整体一致性**同一层级的字体**笔画特征、粗细需保持统一**各个部分做到互相呼应，有所关联，而不是孤立存在或者彼此冲突，**避免出现设计零散 拼凑感强的情况** |  |  |
| 层级清晰 | 通过在标题中使用不同字重的字体，来强调重要层级的内容**精简视觉层级，**去除多余干扰 |  |  |
| 笔画流畅美观 | 确保笔画连接部分**平滑自然**，避免生硬或转角过多 确保**字形的比例协调**一致，避免出现过于压缩或拉伸的情况。合适的字形比例有助于保持整体的流畅性 考虑**字体的疏密变化**，通过笔画粗细、宽窄、轻重、起伏等方面的对比营造出文字的节奏感，确保整体视觉上的流畅性和美观性 |  |  |
| 笔刷明显 | 带笔刷效果的字体比如毛笔字，粉笔字等，需要有明显笔刷效果，还原笔刷真实性 |  |  |
| 画质清晰 | 保证高清画质，100%放大后笔触/元素边缘**清晰锐化**，无模糊感，无锯齿马赛克，无杂色像素 | - | - |
| 易于延展 | 考虑用户体验，提供最大程度的**可编辑性**，如源文件分层细致合理，元素保持完整等，以便素材延展和适配不同场景 | - | - |

### 设计要点说明

| **设计要点** | **说明** | **正面图例** | **反面图例** |  |  |
| --- | --- | --- | --- | --- | --- |
| 通用要点 | 字体结构 | 字形（字体骨骼） | 字形清晰字形结构合理 重心平衡字形间比例关系处理恰当 |  |  |
| 笔形（笔画造型） | 注意笔画样式造型的统一注意笔形的厚度即字重是否匀称美观注意字面占比是否均衡注重笔画的流畅性和连贯性，避免出现生硬或断裂的笔画（特殊造型除外）注意笔画差异性设计，增加字体艺术性和视觉吸引力（利用分解重构、笔画代替、笔画删减、边角修饰、颜值笔画提炼塑造等设计方法） |  |  |  |  |
| 字体效果 | 描边 | 选择与文字对比明显的描边颜色，以**确保文字的可识别性****过粗或过细**的描边可能会影响可读性或视觉效果 |  |  |  |
| 倾斜 | 倾斜角度/方向应保持一致，以确保整体设计的协调性确保倾斜的角度适中，避免过度倾斜导致文字在布局中失去平衡，影响整体的美感和可读性 |  |  |  |  |
| 投影 | 调整投影的方向和距离可以影响文字的立体感和视觉效果调整投影颜色和透明度，避免投影效果看起来过深或模糊 |  |  |  |  |
| 3D | 整体光影、透视、结构与真实情况保持一致 |  |  |  |  |
| AIGC | 字体轮廓清晰 画面元素与主题关联性强肌理类AIGC字体肌理线条流畅连贯，插画3D类AIGC字体元素完整无残缺，无乱码 |  |  |  |  |
| 材质肌理 | 材质或纹理效果可以为文字增添质感，包括不限于真实材质纹理（纸张/布料/金属/砖墙/玻璃/木纹等）、水彩肌理、 水墨肌理、噪点肌理、渐变弥散……**选择符合需求风格的材质/肌理，**不同的材质肌理会赋予文字不同的质感和表现力确保选用的材质/肌理**与字体风格相互协调**在使用渐变或弥散效果时，可以**调整肌理的透明度和混合模式**，以获得更加柔和或独特的视觉效果 |  |  |  |  |
| 字体氛围 | 装饰元素 | 通过**引入视觉元素**，如装饰性线条、插画/3D元素，强化所要表达的情绪和氛围**保持元素的简洁和平衡，**与文字相互衬托**确保装饰元素与文字风格相符****合理安排装饰元素的位置/大小比例，**避免装饰元素与文字重叠或干扰文字的清晰度**保持装饰元素的一致性，**以确保设计风格统一。 |  |  |  |
| 文字排版 | 确保标题字与其他元素的排版结构协调一致，构建整体平衡的视觉效果确保主题突出，利用字体大小、粗细、颜色等元素营造标题字的层次感，突出重点信息通过透视、虚实(景深)、重叠遮挡、立体等手法来塑造字体空间的丰富性 |  |  |  |  |
| 书法字体设计要点 | 注意笔墨运用 | 注意墨色和笔墨运用，通过深浅不同的墨色和运笔的轻重变化，营造出书法作品的立体感和动态美 |  |  |  |
| 笔触清晰 | 笔触的细节（起笔、收笔、飞白等）清晰明确，飞白和笔尖的收尾处理得当，避免模糊或生硬。这样字形的每一个笔画细节都能展示出来，可以保持整体的清晰度和艺术感 |  |  |  |  |
| 笔画流畅 | 单个笔画的起笔、行笔、收笔流畅自然无断裂或粗糙感笔画与笔画之间的过渡平滑连贯不僵硬 |  |  |  |  |
| 强调笔画特点 | 强调书法体的笔画特点，包括笔画的粗细、起落、顿挫等，突出书法作品的独特韵味和表现力 |  |  |  |  |
