# 海报模版品质标准
### 前言

模版品质标准以层次分明、焦点强调、信息传达清晰、简洁明了、和谐统一、主题明确、创新创意、细节丰富、留白得当、易于延展为设计原则；从文字、色彩、图像、排版、背景、材质、光影、空间这8个海报常见要素，进行评定。

### 设计原则及标准

| **设计原则** | 标准 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| **和谐统一** | 海报中的所有要素（包括颜色、字体、图像、光影等）在风格和视觉上保持一致性，形成整体协调的视觉效果。 |  | 问题点：**配色偏灰不协调、灯笼风格/配色不统一**、排版呆板且主次不够分明、不符合潮流氛围感、背景大字融合度差 |
| **层次分明** | 通过大小、颜色、位置等视觉手段，区分开画面的层级，便于观众快速识别信息的优先级。 |  | 问题点：**画面层次杂乱、主体识别度欠缺、底部部分文字看不清** |
| **焦点强调** | 通过设计手法（如大小对比、颜色突出、位置中心化等）突出海报中最重要的信息或视觉元素，吸引观众的注意力。 |  | 问题点：**2025数字不够突出**、排版简陋、场景氛围平淡、齿轮突兀抢视觉、无空间层次感等 |
| **传达清晰** | 确保海报上的文字和图像清晰可辨，字体大小和颜色对比度适宜，以便观众无需过多努力就能阅读和理解内容。 |  | 问题点：**标题字体与背景的颜色对比弱，信息字体的阅读性差，底部元素难以辨别，利益点不突出**，色彩不协调等 |
| **简洁明了***风格化另议 | 去除不必要的元素和装饰，保留最核心的信息和设计元素，使海报内容直接、明确，避免观众产生混淆。 |  | 问题点：**不必要的元素过多、颜色过杂、字体过多、阅读性差**、光影/风格不统一等 |
| **留白得当** | 考虑需求调性，合理利用空白区域，设计匹配的画面气质；同时避免设计过于拥挤，也确保重要元素有足够的空间“呼吸”，增强视觉舒适度。 |  | 问题点：**留白过多不够热闹**、排版不集中、文字对比不明确、蛇形象不够大气 |
| **细节丰富** | 在保持整体简洁的同时，通过精心设计的细节（如图标、装饰性元素等）增加海报的吸引力和深度。 |  | 问题点：**元素与元素间缺乏明暗、投影、材质等细节，排版常规，缺少层次，背景缺少氛围光效。** |
| **创新创意** | 在设计中融入新颖的想法和独特的表现手法，使海报在众多信息中脱颖而出，给人留下深刻印象。 |  | 问题点：**钢笔与灯塔的同构创意不清晰、**画面细节不丰富、排版常规、背景过渡不流畅 |
| **易于延展** | 设计时考虑到海报可能被用于不同的媒介和尺寸，确保核心元素和信息在不同环境下仍能保持编辑的可用性。 |  | 编辑器问题点：**圣诞袜主体元素和礼物/圣诞树等辅助元素，没有和背景进行拆分**源文件问题点：主元素非智能对象图层，后续难复用；图层命名不合理 |

### 设计要点说明

##### 和谐统一

| **要点** | 说明 | 图例 |
| --- | --- | --- |
| 文字 | 选择**与主题和风格相匹配**的字体，来传达情感或强化信息保证**字体间搭配的统一性**，通常来说：黑体配无衬线、宋体配衬线、圆体配圆体考虑受众的语言和文化背景，使用**相匹配的主题文案**，避免引起误解 |  |
| 图像 | 选择合适的图像，**能够直观地传达海报主题**，图形和样式需要适应设计需求通过**统一的图像处理**，包括调色、剪裁、合成等，来保证整体设计的一致注意图像间的**大小比例保持平衡**，与真实情况保持一致（风格化另议，如超现实）描边类型的图像**保持粗细统一**（风格化另议） |  |
| 色彩 | 与**主题或行业特性相匹配**，能够更好的强化信息和情感表达使用**色彩搭配工具或理论来搭配颜色**，使视觉平衡舒适，避免过于刺眼或不协调（脏乱、陈旧）的配色通过**理解色彩心理学的代表含义**，明确主题的情绪氛围，与主题保持统一*以下为特殊情况：若涉及主题文化背景，注意避免使用引起误解或不适的颜色若涉及品牌/行业色相关，应使用品牌色彩或行业色彩，保持视觉识别系统的一致性若涉及线下物料相关，需考虑使用专用色卡来确保色彩的准确性和一致性，避免色差问题 |  |
| 背景 | 和整体**氛围调性保持统一** |  |
| 排版 | 使用版式**重复原则**来保证画面的整齐统一，可以是形式、字号、色彩、对齐方式的统一合理布局海报中的图文信息，**考虑各元素之间的间距和比例**，确保整个海报的视觉平衡感 |  |
| 光影 | 确定主光源的位置，**统一画面各元素的光源方向** |  |

##### 层次分明

| **要点** | 说明 | 图例 |
| --- | --- | --- |
| 文字 | 通过**字号（倍数增长）、字重、字色、投影或立体效果来区分**标题、副标题和正文等信息的层级结构 |  |
| 图像 | 使用**大小、颜色和位置**来梳理图形的层次 |  |
| 通过**图像的虚实、明暗处理**，来提升画面的空间层次 |  |  |
| 色彩 | 通过**颜色的深浅、冷暖**来创建视觉层次，更好的引导观众的视线和注意力 |  |
| 材质 | **添加材质**可以丰富画面层级关系 |  |
| 排版 | 使用**版式亲密性原则**来梳理层级关系，并**用对比原则**来区分整体画面的层次把**文本与符合主题调性的符号/色块/线条/文本背景做结合**，增加画面层次有点线面的处理，进行**动线引导** |  |
| 空间 | 通过在二维空间里**添加物体投影、材质贴图、颜色渐变、虚实对比等方式**，增加二维空间的层次空间感 |  |
| 使用**3D工具制图，并运用好景深工具**，会使得三维空间的真实感更强 |  |  |
| 通过发散构图、螺旋构图、仰视构图、俯视构图、鱼眼镜头和景深镜头等**构图技巧**，可以加强画面的空间层次感 |  |  |

##### 焦点强调

| **要点** | 说明 | 图例 |
| --- | --- | --- |
| 文字 | 通过**颜色、材质、字号大小、字重粗细、字效、添加图形等**方式来强调核心文案信息 |  |
| 图像 | 可通过**醒目的图形、大胆的图像**作为视觉焦点，作为吸引观众注意力的核心区域 |  |
| 色彩 | 使用**强烈的对比配色**技法，来突出关键要传达的信息或视觉元素 |  |
| 光影 | 利用**添加光影或光影技巧**（对比、光源方向、高光反射和阴影效果等）可以有效地突出海报中的关键焦点，吸引观众的注意力 |  |
| 排版 | 通过**强对比的排版原则**来强调焦点 |  |
| 空间 | 处理**好正空间与负空间的关系**，**强调正形的作用**，使其突显，拉开与负空间的距离，以达到引人注目的效果 |  |

##### 传达清晰

| **要点** | 说明 | 图例 |
| --- | --- | --- |
| 文字 | 选择**易读的字体****调整字号字重，间距**，确保在不同的尺寸和距离下都能清晰阅读通过**字色、字效、投影或立体效果**来增加文字与背景的对比 |  |
| 图像 | 保证图像的合理性，**符合认知**图形图片的**主体完整**无残缺，整体造型结构协调使用**高质量、清晰度适中的图片**，放大两倍后无马赛克 | *蛇的身体像虫子，不符合认知 |
| 背景 | 注意**对比**，不影响前景文本或图像的识别 |  |
| 光影 | 遵循**“近亮远暗”原则**，画面光影符合常理，即物体距离离光源越近的部分越亮，越远的部分越暗 |  |

##### 简洁明了

| **要点** | 说明 | 图例 |
| --- | --- | --- |
| 文字 | 文案使用**简短的句子**和**强有力的词汇**，避免冗长和复杂的句式字体选择一般**不超过3个字体**，以免分散观众的注意力 |  |
| 图像 | **避免使用过度复杂/杂乱的图像**，以免分散观众的注意力 |  |
| 色彩 | 避免使用过多的颜色（一般**不超过3种颜色**），以免造成视觉混乱 |  |
| 背景 | **保持简洁**，考虑内容的尺寸延展，需**避免过多的细节** |  |
| 排版 | 使用**排版四大原则（对比/对齐/亲密/重复）**，来保持画面的排版秩序，避免画面混乱 |  |

##### 留白得当

| **要点** | 说明 | 图例 |
| --- | --- | --- |
| 文字 | 注意文字与文字层级间的空间留白，**保持画面的呼吸感** |  |
| 图像 | 涉及图形和图像结合，需**注意保留空间**，避免图像看起来被挤压或剪裁不当 |  |
| 排版 | 需考虑需求调性，利用**版面率知识，创造出相匹配的画面气质**（低版面率-留白多，画面气质更典雅、安静；高版面率-留白少，画面气质更活泼、热闹） |  |

##### 细节丰富

| **要点** | 说明 | 图例 |
| --- | --- | --- |
| 文字 | 使用**填充（颜色/图案）、描边、材质等方式**来丰富字体效果 |  |
| 图像 | 图形的**造型美观线条流畅**，动态不生硬丰富**光影/图案等**细节 |  |
| 排版 | 通过给文本**添加点线面、色块、图形、符号等**元素来丰富细节巧用**相似或反差字体进行搭配**，使版式得到优化正文段落类型，需注意：标点符号避头尾、单字不成行、字距行距、段首缩进（非文章排版）等要点 |  |
| 光影 | 添加**高光和倒角高光**等细节，来刻画出更细腻和真实的光影元素需**考虑环境光**，**投影要干净** |  |
| 材质 | 有**一定的材质、体积刻画**和细节处理 |  |

##### 创新创意

| **要点** | 说明 | 图例 |
| --- | --- | --- |
| 文字 | 关注当下**字体的流行趋势**，结合当下流行的字体混搭或风格化字体，来吸引用户阅读根据作品主题、受众对象的特征，通过**创意文案、创意字体**，来吸引受众对象的关注 |  |
| 图像 | 通过**同构、异构、解构、重构、拟人、矛盾、悖理、变异等创意手法**，创造出创意的画面 |  |
| 色彩 | 注意**色彩流行趋势**，尝试新颖的配色趋势，能够让海报脱引而出 |  |
| 排版 | 在遵循排版原则下，通过**反差、自由的排版方式**来吸引关注 |  |
| 光影 | **把原物体的影子变化为与之相反或相关的另一物体的“影子”**，将其成为作品中突出的创意元素 |  |
| 空间 | 通过**打破真实的正常空间**形式（如异常的造型大小、超现实空间），来获得强烈的视觉冲击使用**非常规构图**，可以提升海报的创意性，创造出引人入胜的视觉故事。使用**超透视**，如：人物/物体：使用透视效果很夸张的人物作为视觉主体，配合人物夸张的动作或面部表情，可以大大加强画面的视觉冲击力和吸引力；超强透视-字体：大字报画面，甚至不需要具体的场景以及过多其他元素，仅通过把主要文字放大，并进行透视扭曲就能得到空间感很强的画面 |  |
