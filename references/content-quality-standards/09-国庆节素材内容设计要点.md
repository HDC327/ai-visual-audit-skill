# 国庆节素材内容设计要点
### 艺术字

在 基础上做的一些补充

| **类型** | **要点说明** | **正面图例** | **反面图例** |
| --- | --- | --- | --- |
| 毛笔艺术字 | 整体有节奏感，字体大小有变化，笔画粗细有变化；毛笔笔触明显，特别注意拖尾效果；金色肌理要明亮精致；构图合理，整体紧凑不松散，视觉上协调稳定；装饰元素品质高，与字体结合度高，无拼凑感； |  |  |
| 数字 | 字形有设计感，图形化，造型细节丰富；数字布局合理，比例平衡；颜色肌理表达准确，如渐变、噪点肌理、金属材质； |  |  |

### 元素

| **类型** | **要点说明** | **正面图例** | **反面图例** |
| --- | --- | --- | --- |
| 鸽子 | 形态需准确，被识别为鸽子，而不是海鸥燕子之类的；动作灵动不呆板，造型有动态感；多只鸽子动作需有区别，避免复制粘贴；构图平衡、错落有致，如对角构图，三点构图等；弥散渐变色彩及材质表达准确不生硬； |  |  |
| 星星 | 星星造型标准化，线条平直，绘图严谨对称；立体感和材质肌理准确表达；多个五角星构图需有规律，一般通过弧线串联； |  |  |

### 背景

| **类型** | **要点说明** | **正面图例** | **反面图例** |
| --- | --- | --- | --- |
| 弥散光感 | 构图留白，有层次感，有视觉焦点；颜色渐变等次丰富，用简便做出体积感、光感；元素组合自然，构成一个整体融合的场景； |  |  |
| 中国风 | 构图留白，有层次感，有视觉焦点；色调明亮，氛围积极；画风及材质表达准确； |  |  |
| 数字场景 | 数字构图协调，识别度高；突出节日氛围感，层次感；数字与场景元素有互动，不孤立或突兀；元素精度高质感强； |  |  |
