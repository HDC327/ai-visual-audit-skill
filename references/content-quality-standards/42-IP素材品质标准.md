# IP素材品质标准
### 设计原则及标准

| 设计原则 | 标准说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 元素完整 | 确保主体元素完整，没有缺失的部分。构图合理裁剪，不影响素材应用。 |  |  |
| 线条流畅 | 线条应该流畅自然，避免出现生硬或断裂的线条。 |  |  |
| 色彩协调 | 色彩搭配和谐，遵循色彩理论，以确保视觉效果的和谐统一。 |  |  |
| 材质细腻 | 准确细腻的的材质表现，适当增加纹理丰富视觉层次感。材质肌理符合实际物体材质。 |  |  |
| 易于识别 | 拟人化设计，轮廓特征具备辨识度，特征需进行强化处理。 |  |  |
| 创意独特 | 需有差异化设定，避免同质化表达。造型或颜色材质需有独特记忆点。 |  |  |
| 场景适用 | 适配多场景应用需求，如线上、线下、衍生品等。 |  | - |
| 画质清晰 | 保证高清画质，100%放大后图形边缘清晰锐化，无模糊感，无锯齿马赛克，无杂色像素。 |  |  |

### 设计要点

| IP类型 | 设计要点 | 要点说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- | --- |
| 人物/机器人 | 表情 | 具备识别度的表情，符合角色当下动作的状态和情绪表达。表情避免呆板木讷，需赋予角色情绪。 |  |  |
| 动作 | 肢体协调，动作具备动态感。可使用特殊视角增加视觉张力。 |  |  |  |
| 服饰 | 服饰材质肌理与实际物理材质贴合。根据主题和风格设计对应服饰。适当添加材质差异的配饰，增加人物细节丰富度，如发饰、眼镜、首饰等。 |  |  |  |
| 比例 | Q版，头身比大于1/3，放大头部。写实版，头身比约1/7.5，接近真人。根据需求和角色设定确定头身比例，可进行适当夸张比例处理，但整体需要协调。 |  | - |  |
| 动物 | 拟人化 | 通过对动物的表情神态，肢体动作进行拟人化处理，表达更多情绪和状态。表情避免呆板木讷，需赋予角色情绪肢体协调，动作具备动态感。 |  |  |
| 特征 | 提取和强调动物特征，如熊猫的黑眼圈、牛的角、兔子的耳朵等，让角色保持识别度。 |  |  |  |
| 植物/物品/符号 | 拟人化 | 通过添加表情、添加肢体手脚等进行拟人化处理，表达更多情绪和状态。表情避免呆板木讷，需赋予角色情绪。肢体协调，动作具备动态感。 |  |  |
| 特征 | 几何抽象变形需保留原本形状的主要特征，让角色保持识别度。 |  |  |  |
