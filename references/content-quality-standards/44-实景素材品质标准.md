# 实景素材品质标准
### 设计原则及标准

| 设计原则 | 标准说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 元素完整 | 确保主体元素完整，没有缺失的部分。构图合理裁剪，不影响素材应用。 |  |  |
| 写实度高 | 准确刻画不同材质质感，如金属反光、水果表皮纹理、织物褶皱等准确呈现主体元素在现实中的造型和特性（特别是AIGC素材格外需注意） |  | 像杨梅 |
| 光影自然 | 通过合理的光源增强立体感，如侧光、柔光，避免过度打光或打光不足导致的失真 |  |  |
| 色彩协调 | 保持元素本身自然色调，避免过度饱和或偏色，食物尤其注意暖色调的食欲感色彩搭配和谐，遵循配色理论，确保应用视觉效果的和谐统一 |  |  |
| 主体突出 | 通过构图和对焦控制，让核心元素成为视觉焦点根据场景需求选择合适的视角，如俯视展示整体布局，45°斜角增强立体感、特写放大细节等 |  |  |
| 细节丰富 | 添加相关符合主体属性的装饰元素细节，增强氛围感，如餐具、水珠、蒸汽等 |  |  |
| 易于合成 | 具有良好的免抠效果，如边缘毛发抠图、主体透明度等表现，在不同背景上都能很好地展示和应用，无需过多调整就能与已有画面内容合成 |  |  |
| 画质清晰 | 保证高清画质，100%放大后图形边缘清晰锐化，无模糊感，无锯齿马赛克，无杂色像素。 |  |  |

### 设计要点

| 实景素材分类 | 要点说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 人物 | 避免僵硬摆拍，捕捉自然表情，肤色和肤质还原需真实，尤其避免过度磨皮 |  | 磨皮严重 |
| 动物 | 毛发及身上肌理处理保留细节，动态场景注意动作角度与肢体比例呈现 |  |  |
| 植物 | 突出自然光线下的叶片通透感，花朵娇嫩感，保留叶脉细节造型优美，画面构成简洁，避免凌乱 |  |  |
| 食物 | 强调新鲜感（水珠、蒸汽）、食材质感（面包焦脆感）、配色对比（绿色香草点缀红色草莓） |  |  |
| 物品 | 强调表面材质，如金属、石头、塑料、纺织物、玻璃等 |  |  |
