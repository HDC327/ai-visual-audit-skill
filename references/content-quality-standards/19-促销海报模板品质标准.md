# 促销海报模板品质标准
该标准基于 ，设计时需一并参考

| **设计原则** | **设计要点** | **说明** | **正面案例** | **负面案例** |
| --- | --- | --- | --- | --- |
| **和谐统一** | **符合受众调性** | **明确目标受众**的年龄、性别、消费习惯、兴趣爱好等，根据目标受众的偏好选择视觉元素、色彩和语言风格。如儿童产品海报多用活泼色彩与卡通形象；运动产品面向健身爱好者，可采用充满活力的画面和动感的字体。 |  |  |
| **色彩契合主题** | **色彩选择要契合促销活动主题与氛围**。如春节促销可用红色、金色营造喜庆氛围；夏季清凉产品促销，可选用蓝色、绿色等冷色调。 |  |  |  |
| **层次分明** | **时间地点明确** | **清晰标注活动时间、地点**，避免消费者产生疑惑。时间可精确到具体日期、时段，地点详细至具体门店地址或线上平台链接。 |  |  |
| **排版层次合理** | 合理**划分海报区域**，将标题、图片、优惠信息、品牌标识等内容有序安排。常见布局有 “上下结构”“左右结构”“中心聚焦” 等，确保信息层次分明。 |  |  |  |
| 卡项类排版，信息内容较多，设计师需要合理分配信息层级，通常情况下需要将优惠产品（产品名称/产品抠图）、优惠信息（促销价格/买X送X）做最高信息层级进行设计。 |  |  |  |  |
| **焦点强调** | **强调主题/利益点** | **促销主题**，如 “周年庆大促”“换季清仓”“双11” 等要在海报中显著展示，让受众**一眼明晰活动主旨**。 |  |  |
| **核心优惠信息**，如 “5折起”“买一送一”“满减金额” 等，以**较大字号、独特颜色或特效突出呈现**，吸引消费者关注。 |  |  |  |  |
| **强调产品/促销元素** | 将**产品**位于画面的核心区域，多产品则需将产品进行陈列布局；通过**色彩对比、投影、材质等方式来突出**产品轮廓、包装等核心信息 |  |  |  |
| 促销素材元素，如"喇叭""礼物盒""购物车/篮""折线箭头"等，在海报核心区域展示，有烘托促销氛围、展示产品等作用。 |  |  |  |  |
| **传达清晰** | **文字清晰易读** | 注意**字体选择，间距保持合理**，保证各层级信息的可读性 |  |  |
| **图像清晰易懂** | **产品图片要清晰展示细节**，能激发消费者购买欲望；也可搭配场景图，帮助消费者更好地理解产品使用场景。 |  |  |  |
| 使用**简洁易懂的图标辅助传达信息**，如用购物车图标表示购买，时钟图标强调活动限时，增强信息传达效率。 |  |  |  |  |
| **留白得当** | **元素布局协调** | 各**元素间保持适当间距**，避免过于拥挤或松散，使海报整体布局平衡、协调。 |  |  |
| **创新创意********** | **文字类创意** | 主题类文字可用**新颖的材质/风格**表现 |  | / |
| **图像类创意** | 将**贴合主题或促销元素进行创意演绎**，如将玩具包装盒、贩卖机作为产品的储物筐 |  | / |  |
| **​** | 将**文字框**做成撕纸、路牌等创意 |  | ​ |  |
| **强视觉冲击** | 使用**超透视和非常规构图**，来制造视觉冲击，吸引消费 |  | / |  |
| **易于延展** | **文字易用** | **文字背景特效还原：**文字+纯色底框色块的设计样式，需在编辑器内使用文字背景功能直接还原。说明：除个性化风格外，需尽量设置还原 |  | / |
| **划重点文字特效还原**：划重点文字需在编辑器内直接还原。说明：除个性化风格外，需尽量设置还原 |  | / |  |  |
| 形文字特效还原：**变形文字需要在编辑器内直接还原**；建议将文字部分空出,在编辑器内完成字体设计*变形文字也包括PS中的斜切文字 |  | / |  |  |
| **列表文字效果还原**：列表文字需在编辑器内直接还原。说明：除个性化风格外，需尽量设置还原 |  | / |  |  |
| **​** | **固定宽高（自动字号）**：固定文本框的大小，但是输入的文字过多，会自动适配缩小以适应当前文本框。 |  | / |  |
| **图像易用** | **矢量图形：**但凡涉及到矢量图形（基础线条/色块/图形/图框/icon等）都需保留矢量可编辑，便于用户修改颜色。**编辑器内需检查是否能修改颜色** |  | / |  |
| **智能拉伸：**凡是遇到文字+规则底框的组合形式，均可选择智能拉伸。选择智能拉伸之后的底框，会根据文字自动缩放底框长度宽度，能提升用户体验。 |  | / |  |  |
| **元素投影层需单独拆分**，在编辑器里保持元素与投影图层合理分层并成组元素效果层若能在编辑器里还原的尽量在编辑器里还原实景图的投影根据实际使用情况在编辑器内还原或单独拆分插画手绘元素的投影根据实际使用情况在编辑器还原或单独拆分 |  | / |  |  |
| **保持完整和独立****可以成为独立个体使用**的元素，需要保持独立图层，主要是方便用户对画面进行一些元素上的移动和调整，保留模板的可编辑性最大化;相隔太远的元素，一定要分割开成为独立的图层，包括背景上的装饰元素，否则也不方便进行元素的移动和调整；元素可以超出画布注意：C4D相关作品也需要遵循元素完整和独立原则，需要分层渲染，尽量做到各元素可单独移动。<br>**单个元素裁切干净**注意元素裁切干净，不要**有多余的元素在看不见的地方**，会导致图层过大，无法导入的问题 |  | / |  |  |
| **背景易用** | **背景需保持简约干净**考虑背景的尺寸延展，背景需保持简约干净，不出现具体的元素，若有具体元素需进行拆分。 |  | / |  |
