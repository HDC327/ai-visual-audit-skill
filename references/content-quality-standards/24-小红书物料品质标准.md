# 小红书物料品质标准
该标准基于 ，设计时需一并参考

### 设计原则及要点说明

| **设计原则** | **要点** | **说明** | **封面-正面案例** | **封面-负面案例** | **配图-正面案例** | **配图-负面案例** |
| --- | --- | --- | --- | --- | --- | --- |
| **和谐统一** | 文字 | **选择与主题和风格相匹配的字体，来传达情感或强化信息**保证字体间搭配的统一性，通常来说：黑体配无衬线、宋体配衬线、圆体配圆体考虑受众的语言和文化背景，使用相匹配的主题文案，避免引起误解 |  |  |  |  |
| 图形/图像 | **选择合适的图像，能够直观地传达海报主题**，图形和样式需要适应设计需求通过统一的图像处理，包括调色、剪裁、合成等，来保证整体设计的一致注意图像间的大小比例保持平衡，与真实情况保持一致（风格化另议，如超现实）描边类型的图像保持粗细视觉上统一（风格化另议） |  |  |  |  |  |
| 色彩 | 与**主题或行业特性相匹配**，能够更好的强化信息和情感表达使用**色彩搭配工具或理论来搭配颜色**，使视觉平衡舒适，避免过于刺眼或不协调（脏乱、陈旧）的配色通过**理解色彩心理学的代表含义**，明确主题的情绪氛围，与主题保持统一 |  |  | / | / |  |
| 排版 | 使用版式重复原则来保证画面的整齐统一，可以是形式、字号、色彩、对齐方式的统一合理布局图文信息，考虑各元素之间的间距和比例，确保整体的视觉平衡感 |  |  |  |  |  |
| 光影 | 确定主光源的位置，统一画面各元素的光源方向 |  |  | / | / |  |
| **层次分明** | 文字 | 通过**字号、字重、字色、投影、字效****、添加图形**等来区分标题、副标题和正文等信息的**层级结构** |  |  |  |  |
| 图形/图像 | 使用**大小、颜色、位置、层级、虚实**来梳理图形的层次 |  |  |  |  |  |
| 色彩 | 通过**颜色的深浅或冷暖**来创建视觉层次，更好的引导观众的视线和注意力 |  |  |  |  |  |
| 排版 | 使用**版式亲密性原则**来梳理层级关系，并用**对比原则**来区分整体画面的层次把**文本与符合主题调性的 符号/色块/线条/文本背景 做结合**，增加画面层次结合**点线面**的处理，进行 **动线引导** |  |  |  |  |  |
| **焦点强调** | 文字 | 通过**颜色、材质、字号、字重、字效、添加图形**等方式来**强调核心文案**信息 |  |  |  |  |
| 图形/图像 | 可通过**醒目的图形、大胆的图像**作为视觉焦点，作为吸引观众注意力的核心区域 |  |  |  |  |  |
| 色彩 | 使用**强烈的对比配色**技法，来突出关键要传达的信息或视觉元素 |  |  | / | / |  |
| 排版 | 通过**强对比的排版原则**来强调焦点 |  |  |  |  |  |
| **传达清晰** | 文字 | **选择易读的字体**调整字号字重，间距，确保在不同的尺寸和距离下都能清晰阅读通过字色、字效、投影或立体效果来增加文字与背景的对比 |  |  |  |  |
| 图形/图像 | 保证图像的合理性，**符合认知**图形图片的**主体完整**无残缺，整体造型结构协调使用**高质量、清晰度高的图片** |  |  |  |  |  |
| 背景 | **避免使用过度复杂/杂乱的图像**，以免分散观众的注意力注意**对比**，不影响前景文本或图像的识别 |  |  |  |  |  |
| 光影 | 遵循**“近亮远暗”原则**，画面光影符合常理，即物体距离离光源越近的部分越亮，越远的部分越暗 |  |  | / | / |  |
| **留白得当****** | 文字 | 注意文字与文字层级间的空间留白，保持画面的呼吸感 |  |  |  |  |
| 图形/图像 | 涉及图形和图像结合，需注意保留空间，避免图像看起来被挤压或剪裁不当 |  |  | / | / |  |
| 排版 | 需考虑需求调性，利用**版面率知识，创造出相匹配的画面气质**高版面率-留白少，画面气质更活泼、热闹 |  |  |  |  |  |
| 低版面率-留白多，画面气质更典雅、安静 |  |  |  |  |  |  |
| **细节丰富** | 文字 | 使用填充（颜色/图案）、描边、错落、变形、材质等方式来丰富字体效果 |  |  | / | / |
| 排版 | 通过给文本**添加点线面、色块、图形、符号等**元素来丰富细节巧用**相似或反差字体进行搭配**，使版式得到优化正文段落类型，需注意：标点符号避头尾、单字不成行、字距行距、段首缩进（非文章排版）等要点 |  |  |  |  |  |
| **创新创意** | 文字 | 关注当下字体的流行趋势，结合当下**流行的字体混搭或风格化字体**，来吸引用户阅读根据作品主题、受众对象的特征，通过**创意文案、创意字体**，来吸引受众对象的关注有**多材质、体积刻画**和细节处理 |  |  | / | / |
| 图形/图像 | 通过**同构、异构、解构、重构、拟人、矛盾、悖理、变异、多材质等创意手法**，创造出创意的画面 |  |  |  |  |  |
| 色彩 | 注意**色彩流行趋势**，尝试新颖的配色趋势，能够让海报脱引而出 |  |  | / | / |  |
| 背景 | 尝试**新颖的****滤镜**，让海报更具特色 |  |  | / | / |  |
| **画中画**形式让画面更丰富吸睛 |  |  | / | / |  |  |
| 排版 | 在遵循排版原则下，通过**反差、自由的排版方式**来吸引关注 |  |  |  |  |  |
| 通过**非常规的图框/拼图方式**使画面更加新颖 |  |  | / | / |  |  |
| **易于延展** | 文字 | **划重点、列表样式、文字背景等**元素需使用编辑器进行还原，保证编辑可用性 |  |  |  |  |
| 图形/图像​ | **图片容器、描边等**图像类型需使用编辑器进行还原，保证编辑可用性由于app端暂不支持**拼图组件功能，**因此上传时**暂不使用**拼图组件 |  | / | / |  |  |
| 对于有小眼睛的的元素，务必单独拆分，无需使用的用户可以快速删除 |  | / | / |  |  |  |
| 色彩 | 矢量图形配色保持可编辑 |  | / | / |  |  |
| 背景 | 纯色或渐变背景配色保持可编辑 |  | / | / |  |  |
| 保持简洁，考虑内容的尺寸延展，**背景元素需尽量拆分** |  |  |  |  |  |  |
| 排版 | **合理分组**，同层级文案或临近元素打组 |  |  |  |  |  |
