# 产品营销海报模板品质标准
该设计要点基于，设计时需一并参考

| 设计原则 | 设计要点 | 说明 | 正面案例 | 负面案例 |
| --- | --- | --- | --- | --- |
| 和谐统一 | 符合受众调性 | **明确目标受众**的年龄、性别、消费习惯、兴趣爱好等，根据目标受众的偏好选择视觉元素、色彩和语言风格。比如母婴行业一般都是可爱的婴儿形象、温馨的亲子互动画面。科技行业一般都是3D建模、光效和动态效果，增强未来感。餐饮行业一般使用高饱和度的食物图片，激发食欲。 |  |  |
| 色彩契合主题 | **色彩选择要契合行业产品营销主题与氛围**。如以科技产品为例，多选用蓝色（科技、理性与未来感）、银色（金属质感与精致工艺）等冷色调。食品类产品，暖色调运用较为广泛（红、橙激发食欲，传递美味、热情与活力） |  |  |  |
| 层次分明 | 空间位置引导 | **视觉路径引导**：利用线条、箭头、人物视线或其他引导元素，将观众的注意力自然地引向焦点。例如大小/数字引导观众从产品图片看向产品名称和卖点。 |  |  |
| 排版层次合理 | 合理**划分海报区域**，将标题、图片、产品功能点、品牌标识等内容有序安排。常见布局有 “上下结构”“左右结构”“中心聚焦” 等，确保信息层次分明。 |  |  |  |
| **分组与关联**：将相关的信息元素进行分组，形成清晰的视觉单元，便于观众阅读和理解。 |  |  |  |  |
| 焦点强调 | 强调产品卖点 | **核心卖点**：产品的核心卖点是吸引消费者的关键因素，需要在海报中突出强调。比如新品推广，核心焦点可能是产品名称或独特卖点。 |  |  |
| 突出产品 | **黄金分割法则：**将**产品**位于画面的核心区域，多产品则需将产品进行陈列布局；通过**大小对比、色彩对比、投影、材质等方式来突出**产品轮廓、包装等核心信息 |  |  |  |
| **重复与强化**：对于重要的信息，可以通过适当重复或在不同位置进行强调，增强观众的记忆效果。以**较大字号、独特颜色、形状或特效突出呈现**，吸引消费者关注。 |  |  |  |  |
| 传达清晰 | 文字清晰易读 | 注意**字体选择，间距保持合理**，保证各层级信息的可读性 |  |  |
| 图像清晰易懂 | 使用高质量、高分辨率的图像，避免模糊或失真的图片。图像应与主题紧密相关，能够直观地传达产品或活动的核心内容。 |  |  |  |
| 使用简洁、直观的图形和图标来辅助信息传达，帮助观众快速理解复杂的概念或数据。图形和图标应与整体设计风格保持一致。 |  |  |  |  |
| 场景关联 | 场景应与产品特性紧密相关，直观展示产品的优势。晾晒的衣物模拟日常家庭洗涤场景暗示了产品的使用效果 |  |  |  |
| 留白得当 | 元素布局协调 | 各**元素间保持适当间距**，避免过于拥挤或松散，使海报整体布局平衡、协调。 |  |  |
| 创新创意 | 文字类创意 | 主题类文字可用**新颖的材质/风格**表现 |  | / |
| 图像类创意 | 通过添加热点/独特的创意元素/人物来吸引消费者的注意力，使海报在众多竞品中脱颖而出。 |  | / |  |
| 将**文字框元素**做成毛绒、撕纸等创意 |  |  |  |  |
| 易于延展 | 文字易用 | **文字背景特效还原：**文字+纯色底框色块的设计样式，需在编辑器内使用文字背景功能直接还原。说明：除个性化风格外，需尽量设置还原 |  | / |
| **划重点文字特效还原**：划重点文字需在编辑器内直接还原。说明：除个性化风格外，需尽量设置还原 |  | / |  |  |
| 变形文字特效还原：**变形文字需要在编辑器内直接还原**；建议将文字部分空出,在编辑器内完成字体设计*变形文字也包括PS中的斜切文字 |  | / |  |  |
| **列表文字效果还原**：列表文字需在编辑器内直接还原。说明：除个性化风格外，需尽量设置还原 |  | / |  |  |
| 图像易用 | **矢量图形：**但凡涉及到矢量图形（基础线条/色块/图形/图框/icon等）都需保留矢量可编辑，便于用户修改颜色。**编辑器内需检查是否能修改颜色** |  | / |  |
| **智能拉伸：**凡是遇到文字+规则底框的组合形式，均可选择智能拉伸。选择智能拉伸之后的底框，会根据文字自动缩放底框长度宽度，能提升用户体验。 |  | / |  |  |
| **元素投影层需单独拆分**，在编辑器里保持元素与投影图层合理分层并成组元素效果层若能在编辑器里还原的尽量在编辑器里还原实景图的投影根据实际使用情况在编辑器内还原或单独拆分插画手绘元素的投影根据实际使用情况在编辑器还原或单独拆分 |  | / |  |  |
| 背景易用 | 考虑物料尺寸调整的使用场景，背景最好保持简约干净，不出现具体的元素，若有具体元素需进行拆分 |  | / |  |
