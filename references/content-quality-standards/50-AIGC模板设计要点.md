## 一.质量标准

### A.正确性

**您在AIGC成图中，若遇到一些问题，需先对图像进行修复或自审，方可进行上传审核。**

| **分类说明** | **具体标准** | **错误案例** |
| --- | --- | --- |
| 人物 | **人体如四肢，五官面部符合常规认知**不可出现肢体残缺、关节错位、面部崩坏、比例失调等情况不可出现同真实人物、知名游戏、动漫IP人物相同，相似，神似等情况 |  |
| 动物 | **动植物造型符合常规认知，能清晰识别品种**写实类动物不可出现肢体增生、品种组合、比例异常等情况；植物不可出现品种嫁接，花/叶/果不同种类 |  |
| 物品 | **物品造型符合常规认知，能清晰识别物品类型**几何形体物品线条流畅，不可出现细节变形；方/圆造型需规整 |  |
| 建筑 | **建筑结构清晰，直线规整，曲线流畅**能清晰识别建筑廊柱、屋檐等内容，非模糊效果设计手法的画面需能清晰识别建筑主体及轮廓 |  |
| 光效 | **线性光效需具备流畅度与连续性，点状光效分布均衡，造型完整**不可出现光效元素不一致，线元素堆叠、杂乱、断开等情况 |  |

## 二.生产要点

### A.生产类型限制

**限制说明：大部分场景下，AIGC模版都需要生成****≥2个以上元素进行组合/合成，主要考虑以下两点：**

1.为了稿定模版的**安全性**：虽然当前法律暂无针对AI作品权利归属的相关约定，但稿定评估直接生成的AI素材还是存在较高的侵权风险，因此需要通过多图层生产限制来规避风险。

2.持续保障用户编辑的**易用性**：AIGC模版和非AIGC模版的主要区别在于生产工具、素材质量和制图效率，因此对AIGC模版的分层要求不会有太大的变化，**生产时依旧需要尽可能保证素材是方便用户挪动的**。
