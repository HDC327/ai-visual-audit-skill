# 直播物料品质标准

### 设计原则及标准

| **设计原则** | 标准 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| **营造氛围** | 通过适当的**装饰元素**来表达画面主题氛围 有细腻的**光影效果**，精准的**材质表现**，让画面有精致的质感氛围 |  |  |
| **有空间感** | 通过合理组合画面元素的前后层次关系，运用色彩明暗对比与光影虚实效果处理，增强画面纵深感，塑造层次分明富有空间感的直播背景 |  |  |
| 构图平衡 | 构图保持**视觉平衡**，无论是对称还是非对称，都应给人**稳定和有序**的感觉 |  |  |
| **层级清晰** | 通过合理的排版、视觉引导与信息分级，打造简洁有序的视觉结构 核心信息（如直播主题、活动折扣）置于视觉中心，次级信息（如活动细则）放在次要区域，背景和装饰信息不干扰主视觉 |  |  |
| **主体/利益点突出** | 通过强化字体设计、突出色彩对比、增加图形符号辅助等方式**突出主标题、价格和优惠力度等内容****精简视觉层级，**避免画面拥挤或焦点分散动态海报：使用局部动画突出重要信息，如价格/利益点跳动或倒计时闪烁 （常用与背景） |  |  |
| **色彩协调** | 色彩**搭配和谐**，遵循色彩理论，如互补色、相近色等，使视觉感受舒适根据行业调性或节日主题调整配色风格，营造适宜的氛围 |  |  |
| **简约留白** | 避免画面过于拥挤，保证主播/商品展示区的留白并确保核心信息（如优惠力度）清晰呈现 直播贴片利用边缘或内容间的留白来分隔不同层级信息，使画面看起来更干净整洁 |  |  |
| 画质清晰 | 保证高清画质，100%放大后图形边缘**清晰锐化**，无模糊感，无锯齿马赛克，无杂色像素 | - | - |

### 设计要点说明

| 要点 | 说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| **素材设计要点** | **展台类元素**应靠近画面中心下方（作为下贴片），周围留白适当，以确保预留的商品位置突出且空间充裕，展台上方无干扰元素 |  |  |
|  | **底框类元素**的风格需契合主题，颜色需与文字形成对比，确保信息可读性 |  |  |
|  | **光效类元素**的光源方向一致，营造真实感和整体感 可通过光效引导视线集中于商品、标题或关键信息区域 清晰细腻，边缘过渡柔和自然，避免模糊粗糙或失真 科技、梦幻场景中可适当加入动态光晕或粒子效果 |  |  |
| **贴片设计要点** | 信息传达：贴片的主要任务是快速传递信息，如优惠活动、促销信息、抽奖提示等，设计时应以**简洁、清晰为优先原则** |  |  |
|  | 贴片造型需基于直播主题展开设计，结合自然、几何、品牌和节日元素，运用独特轮廓与细节处理，打造符合直播场景的贴片形状，以增强视觉记忆点和画面趣味性避免过于复杂的轮廓，保持清晰易识别的造型 注重形状创意与功能结合，做到造型独特的同时保证信息清晰、与场景协调 |  |  |
| 字体设计要点​ | 字体风格需**匹配主题氛围，**如常规促销活动选择醒目、富有冲击力的加粗无衬线体；国风场景使用书法体、国潮风字体；趣味场景采用手写体、卡通类字体 |  |  |
| 字体设计需考虑背景风格和配色，做到和背景、元素协调统一 |  |  |  |
| 重点信息的字体需足够醒目，**避免使用过细的字体****字号层级划分清晰 ，**确保核心信息突出，细则信息（如活动规则）不占据过多视觉空间 |  |  |  |
| 色彩搭配要点 | **贴合主题，**如促销活动用红橙暖色营造热烈氛围，科技产品倾向于蓝色和灰色体现专业感，春夏促销则需使用轻快明亮的色彩来契合季节感受 |  |  |
| **和谐统一**，色彩数量控制在3-5种，冷暖色调搭配协调，避免过多颜色导致画面杂乱 |  |  |  |
| **对比强化重点**，利用明暗、冷暖或高低饱和度对比突出信息。如浅色背景配深色文字，深色背景用亮色文字，提升可读性 |  |  |  |
| 背景贴片要点 | **背景留白:**直播间需优先预留主播核心展示位，背景设计需**避免装饰元素、道具或图案过度堆砌**，刻意**保留充足视觉留白**。这样既能让主播成为画面焦点，避免观众注意力被杂乱背景分散，也能让整体视觉更清爽通透，提升直播观感的整洁度与专业感。 |  |  |
