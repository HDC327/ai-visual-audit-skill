# 卡片模板品质标准
该设计要点基于，设计时需一并参考

### 基础视觉标准

| **设计原则** | **设计要点** | **说明** | **正面案例** | **负面案例** |
| --- | --- | --- | --- | --- |
| **和谐统一** | 文字 | **选择与主题和风格相匹配的字体，来传达情感氛围或强化信息**梦幻感、唯美感、酸性小卡通常选择华丽的称线字体潮流感扁平风小卡选择粗大的无称线字体可爱感甜美感小卡可选择圆润可爱一些的字体混搭字体需注意混搭匹配度，可加一些点、线、面装饰缓解平衡感 | 梦幻感、唯美感潮流感可爱感、甜美感 |  |
| 图像 | **选择合适的图像，能够直观地传达海报主题**使用年轻潮流有“网红感”的人物素材，可在素材库搜索关键词“网红”获取通过**统一的图像处理**，来保证整体设计的一致图像描边**保持粗细统一** |  |  |  |
| 色彩 | 与**主题或行业特性相匹配**，能够更好的强化信息和情感表达如：黑粉甜酷，黑红冷酷，黄绿清新等，配色上需与人物风格和谐统一小卡多以华丽为主，无特殊要求配色上尽量避免使用纯黑白灰的配色 |  |  |  |
| 背景 | 和整体**氛围调性保持统一**常用背景：光感氛围、纯色/渐变/色块切割背景、网格背景 |  |  |  |
| **层次分明** | 文字 | 通过**字号、字重、字色、字效****、添加图形**等来区分主副标题的**层级结构** |  |  |
| 排版 | 使用**版式亲密性原则**来梳理层级关系，并用**对比、层级**来区分整体画面的层次 |  |  |  |
| **焦点强调** | 文字 | 通过**颜色、材质、字号、字重、字效、字体混搭、添加图形**等方式来**强调核心文案**信息 |  | / |
| **传达清晰********** | 图像 | **清晰的抠图、适当的裁剪比例，**突出人物脸部，取腰部或胸部以上 |  |  |
| **简洁明了****** | 文字 | 文案使用**简短的句子**和**强有力的词汇，直接传达核心信息，**避免过多文字堆砌，保持简洁明了；小卡文字多以英文为主，字体选择一般**不超过3个字体**，以免分散观众的注意力 |  | / |
| **留白得当** | 排版 | 需考虑需求调性，利用**版面率知识，创造出相匹配的画面气质**低版面率-留白多，画面气质更典雅、安静；高版面率-留白少，画面气质更活泼、热闹 | 低版面率 — 安静高版面率 — 活泼 | / |

### 优质视觉标准

在基础视觉标准之上，增加要求。

| **设计原则** | **设计要点** | **说明** | **案例** |
| --- | --- | --- | --- |
| **细节丰富** | 文字 | 使用填充（颜色/图案）、描边、错落、倾斜、变形、手写字、添加装饰等方式来**丰富字体效果** |  |
| 排版 | 通过给文本**添加点线面、色块、图形、符号等**元素来丰富细节，使版式得到优化 |  |  |
| **​** | 图形/图像 | 根据作品主题，使用丰富饱满的元素增加视觉氛围，人物增加发夹、首饰、猫耳朵、腮红等丰富画面细节 |  |
| **创新创意** | 排版 | 根据作品主题、人物特征，**巧用相似或反差字体进行混搭、特殊光效处理**，来增加视觉氛围 |  |
