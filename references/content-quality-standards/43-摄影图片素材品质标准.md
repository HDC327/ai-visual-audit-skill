# 摄影图片素材品质标准
### 设计原则及标准

#### 通用摄影图片品质标准

| 设计原则 | 标准说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 构图平衡 | 主体在画面中位置恰当，构图稳定、视觉平衡，避免偏重一侧或头重脚轻。如风光的三分法、人像的居中或对角线构图。 |  |  |
| 光线合理 | 光源方向自然，整体亮度适中，有足够明暗对比呈现层次。无过曝死白或阴影死黑，强调质感与氛围表达。 |  |  |
| 对焦清晰 | 关键主体应对焦准确，边缘清晰无虚焦。整体图像应无因抖动或错误设置导致的模糊。 |  |  |
| 色彩协调 | 色调统一、色彩自然不过饱和，主配色搭配合理。既能还原真实物体色彩，又支持营造氛围（如冷暖对比、暗色背景突显亮色主体）。 |  |  |
| 主体明显 | 图像应有明确主体，观者一眼能理解画面重点或讲述的内容。主题不被杂乱背景、颜色冲突或无关物体干扰。 |  |  |
| 背景简洁 | 背景需简洁、层次清晰，避免杂物或噪点干扰。在风光、室内、静物等摄影中，背景应辅助而非抢镜。 |  |  |
| 画质清晰 | 分辨率高、细节保留良好，无明显压缩、失真、噪点或伪影，适用于从社媒展示到商业印刷的各类输出标准。 |  |  |

#### AI生成摄影图片品质标准

AI生成摄影图片除了应遵循通用摄影图片品质标准外，因生产方式特殊性，做如下补充

| **设计原则** | **标准说明** | **正面图例** | **反面图例** |
| --- | --- | --- | --- |
| 真实感强 | 图像应符合现实中的物理规律与视觉经验，例如光影自然、动作合理、材质可信。整体观感不应让人一眼识别为“假图”。 |  |  |
| 光影协调统一 | 所有阴影、反光和明暗应基于同一光源影响，避免人物、物体、背景出现光源不一致的错乱现象。 |  |  |
| 结构造型准确 | 人物五官、手指、身体比例符合解剖常识，静物形状对称正常，建筑线条不应断裂或融合。 |  |  |
| 纹理细节自然 | 食物应有真实质感（如油光、水汽），建筑应有材质清晰感（如砖纹、玻璃反光），服饰应具褶皱与材质差异。 |  |  |
| 场景逻辑正确 | 图中元素需空间关系合理，物体不漂浮、不融合；环境、动作、道具的组合应具现实合理性。 |  |  |
| 尺寸比例准确 | 人、动物、物体自身比例，元素与其环境的比例符合常识，如桌子不高过人、盘子不小于杯子。 |  |  |
| 文本清晰正确 | 场景中出现的文字应清晰可读，无乱码、错别字或拼写错误。 |  |  |
| 风格统一稳定 | 组图在光感、色调、构图风格上应保持一致，适用于品牌图集、广告项目或内容系列。 |  |  |
| 主体一致性稳定 | 同一人物、动物、产品在多图中应保持外观、服饰、结构一致，避免“同人不同脸”或道具变形问题。 |  |  |
| 人物肖像多样性 | 在合照中，不同人物之间的长相应该有所区别（双胞胎除外），避免“多人共享同一张脸”的诡异情况。 |  |  |

### 设计要点

| 摄影图片分类 | 要点说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 人物 | **造型结构准确：**面部五官、肢体比例自然无畸变，表情真实、姿态协调。**光影方向统一：**人物与背景光线方向一致，阴影自然不过度夸张。**手部细节正确：**手指数量与动作姿势应符合常规，无融合、重复或扭曲。**皮肤纹理自然：**肤色均匀，毛孔、皱纹等纹理真实不过度美化或“塑料感”。**模特外貌优秀：**模特的外貌符合主流审美取向，令人观感舒适，表情眼神能准确传递情绪和氛围。**人物一致性：**若为系列图，人物发型、服装、肤色与面部结构应保持统一。**肖像权合规：**避免生成逼真但虚构的公众人物/名人肖像，严禁模仿现实人物外貌误导观众或侵犯个人肖像权； |  |  |
| 动物 | **结构比例正确：**头部、四肢比例自然，姿态协调，无重复器官或异常连接。**毛发与纹理真实：**毛色分布自然、眼神有神、毛发过渡平滑无块状杂质感。**背景与环境合理：**所处环境符合动物生存习性（如森林、草原、水边），避免动物“悬空”或环境错误。**动作自然：**行为姿态真实可信，避免产生不符合动物习性的夸张姿势。**物种识别准确：**外形、毛色、耳型、体型等应与真实品种特征一致；尤其用于科普、教育、出版时，不得混淆不同动物种类（如混淆犬种、猫科、鸟类等），避免误导。 |  |  |
| 自然风光 | **构图层次分明：**前景、中景、远景分布合理，营造空间感与纵深。地平线应保持水平。**光线自然统一：**日出/日落/阴天等光源方向一致，云层、山体、倒影与光影不冲突。**画面细节清晰：**景深范围广，山石、草地、水面等元素应细节丰富，避免涂抹感。**色彩自然不过饱和：**蓝天、绿地等应避免过度艳丽，保持真实感。适当保留自然雾霭或光晕。**地貌结构合理：**山体、岩石、河道等地貌形态应符合地理特征，不应生成无法存在的地形（如“漂浮山”或“断崖叠加”）。 |  |  |
| 城市风光 | **建筑线条准确：**建筑垂直、透视自然，窗户排列整齐无断裂或融合。**夜景光控得当：**高光不过曝，灯光真实，有阴影和层次。**街道细节清晰：**地面、标识、路灯、交通标线清楚，画面干净。**人车比例协调：**街景中人物和车辆体积应合理，符合城市实际比例关系。**城市标识清晰：**公共标牌、街道名、店铺招牌如出现，应语义正确、排版自然。**地标造型准确：**出现城市地标时，造型应准确无误，符合实际地标外观和特点。 |  |  |
| 室内空间 | **透视正确：**墙面、地板、天花线条平直，空间结构自然。**布置合理整洁：**家具陈设符合生活习惯，无漂浮、错位、重叠。**光线柔和均匀：**自然光与灯光结合，照明合理不过曝，无阴暗死角。**材质纹理清晰：**地板、沙发、玻璃、金属等应有对应质感细节。**功能区域合理：**卧室、厨房、客厅等布局应符合生活功能逻辑，不混搭（如水池出现在卧室）。**家居物品真实可用：**椅子能坐、柜门有把手、灯具构造完整，避免生成抽象家具或错位物品。 |  |  |
| 美食 | **主菜突出：**食物主体居视觉焦点，配菜/道具不抢戏。**光线表现质感：**侧光或背光提升食物立体感、湿润感或油光。**色彩诱人自然：**色调温暖协调，食材新鲜，食物本色不过度修饰。**背景干净衬托：**背景布、餐具应与食物风格一致，避免杂乱。**食物结构准确：**避免生成不符合现实食物形态的错误图像。**餐具使用逻辑：**餐盘、筷子、杯子尺寸匹配，摆放合乎餐桌逻辑，无重影或穿插。 |  |  |
| 静物 | **构图简洁有设计感：**遵循极简或美学构图规则，避免堆砌和混乱。**光影强调质感：**材质反射自然，如陶瓷、木纹、金属等应真实可感。**产品边缘锐利：**对焦准确，清晰展现形状与结构细节。**背景统一协调：**使用纯色或渐变背景，色调不喧宾夺主。**文字图案清晰：**如果出现产品包装，字体应清晰无乱码，排版符合实际印刷风格。**产品功能合理：**产品设计应符合逻辑，如按钮位置、构造合理。 |  |  |
