# 【AIGC】模板创作-案例教程
## MJ案例

**MJ常用功能：高清放大、局部重绘、扩图**

### 局部重绘 案例

选取1张图用【局部重绘】的功能进行画面拆分再组合（快速拆分图层常用方法）

| 模板展示 | 根据构思填写关键词得到整体画面通过多次生成选出合适的画面 | **☆使用局部重绘**得到背景（删除需要去除的前景关键词） | 抠出元素微调修复​，添加文案完成排版 |
| --- | --- | --- | --- |
|  |  |  | 母女对望、飘逸的裙子、飞鸟位置调整、爱心落日 |
|  |  |  | 调色去黄、添加粽子素材 |
|  | 关键词1：温馨暖色调画面，妈妈抱着孩子，她们被粉色黄色白色花朵围绕，花朵环绕成爱心的形状，色彩柔和。重绘关键词2：温馨暖色调画面，粉色黄色白色花朵环绕成爱心的形状，色彩柔和。 |  |  |

### 合成 案例

使用一组或多组关键词生成多张图，然后选取2张以上的图进行合成

| 模板展示 | 根据构思填写关键词--得到整体画面，通过多次生成选出两张以上合适的画面 | 将两张图进行融图，得到新的背景图 |  |
| --- | --- | --- | --- |
|  |  |  | （ps：装饰元素单独生成或选取素材库素材，素材库素材若与画面风格不统一，可先贴到图上后再进行图生图，使风格统一后再抠图使用） |
|  |  |  | 将两个素材合成为一个场景，添加小牛、燕子之类的装饰元素丰富画面 |

### 多元素组合 案例

使用2组以上不同的关键词-生成不同的元素-再进行组合处理

| 模板展示 | 根据需求确定画面元素，使用不同的内容关键词，相同的风格关键词，分别生成不同的元素 | 后期通过抠图、修图、调色、组合完成排版 |  |
