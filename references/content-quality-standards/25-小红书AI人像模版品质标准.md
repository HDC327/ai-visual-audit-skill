### 品质标准

##### 视觉标准

| **设计要点** | **要点说明** | **正面案例** | **负面案例** |
| --- | --- | --- | --- |
| **契合主题** | 围绕单一核心主题，无论是信息文案还是人像表达都需与主题相符，方便用户识别笔记意图。 |  |  |
| **信息清晰** | 文案简洁明了，文字需与画面融合，字体大小按 “核心信息> 辅助信息进行区分，颜色选择与画面对比适中的色系，位置放在画面边缘或空白处，不遮挡人物关键部位。 |  |  |
| **画质清晰** | 放大2倍无明显锯齿 |  |  |
| **焦点突出** | 人像笔记的核心是 “人”，需通过强化人物效果（如描边、滤镜）、构图（如背景虚化、前景遮挡、居中裁剪）弱化无关元素（杂乱的背景、多余的道具），让观众视线第一时间落在人物的表情、穿搭或动作上。 |  |  |
| **有故事感** | 避免呆板的证件照式自拍，用动态抓拍（如大笑、转身、低头浅笑）、服装道具、肢体动作、场景化构图（如在咖啡馆看窗外、在街头奔跑）传递情绪，让用户看到封面就好奇 “她在做什么”。 |  |  |
| **避免过度完美** | 小红书用户偏爱 “真实感”，避免过度 P 图（如磨皮到五官模糊、腿拉到变形），保留自然的皮肤纹理、轻微的瑕疵（如小雀斑、笑纹），让用户觉得 “这是普通人也能实现的美”。 |  | 影棚感、假人感(AI味) |
| **色彩搭配** | 注意色彩的对比和协调性，契合主题调性，避免灰、暗、脏。 |  |  |
| **差异化/吸睛** | 通过鲜明色彩、独特构图/视角、空间层次、差异化观点，快速抓住眼球 。 |  |  |
| **多人互动** | 若笔记涉及多人，需体现在同一个主题场景下，并且需要展示出人物关系和互动性。 |  |  |
| **抠图精细** | ​ |  |  |

| 教育 | **真实场景**：多在教室、户外、家庭等实际教育场景中，增强内容真实感与关联性。**多元身份**：涵盖教师、家长、学生等角色，从不同视角输出教育观点与经验。**互动性强**：有师生互动、亲子互动画面，体现教育过程中的交流与陪伴。**表情神态**：或自信分享（如讲解教育方法）、或亲切沟通（如家校互动场景 ），传递专业又接地气的氛围。**动作关联主题**：通过手势辅助讲解、展示资料等动作，强化教育内容的表达。**风格多样**：既有专业知识输出的严谨感，也有留学、亲子类笔记的轻松感，适配不同教育细分领域 |  |
| 母婴 | **真实亲子互动**：多展现父母与孩子亲昵互动场景，如抱娃、陪玩等，传递温馨育儿氛围。**场景多元**：覆盖居家、户外、购物等场景，体现母婴生活日常，增强内容实用性。**表情自然**：人物表情以微笑、柔和为主，传递积极育儿情绪，拉近距离感。**身份多样**：包含妈妈、爸爸等不同育儿角色，呈现多元育儿视角。**突出宝宝**：宝宝多为画面焦点，通过可爱神态、萌趣动作，吸引关注，强化母婴主题。**动作关联育儿**：有教学、展示母婴用品使用等动作，贴合知识分享、好物推荐的笔记需求。**风格接地气**：画面风格日常化，减少精致滤镜，凸显真实育儿生活，引发共鸣 。 |  |
| 家居 | **场景融入**：人像多自然融入家居空间，如客厅、房间等，与家装、家具等场景深度结合，强化 “家” 的环境关联。**情绪传递**：常展现积极情绪，像分享新家的喜悦、对装修成果的自豪，通过笑容、手势传递温馨、兴奋等感受 。**互动展示**：有与家居物品的互动，比如展示家具、指向装修细节，辅助讲解家居内容，增强实用性分享感。**真实生活化**：风格接地气，以日常状态呈现，少精致摆拍，凸显家居生活的真实模样，引发用户共鸣。**群体多样**：涵盖单人分享、家人 / 朋友共同出镜，体现独居、家庭等不同家居场景下的人像状态 。 |  |

| 情感 | **神态传递情绪**：表情或深沉、或感怀，贴合文字传递的情感主题，像对原生家庭、人生感悟内容，展现出思考、共情神态 。**场景适配主题**：背景简洁或与情感话题关联，如探讨抑郁用素净环境，聊家庭创伤用有故事感场景，强化氛围 。**动作辅助表达**：手势、姿态自然且呼应内容，比如分享感悟时手部轻抬，增强情感传递的感染力 。**风格真实质朴**：不过度修饰，以生活化、真实的形象呈现，拉近与读者距离，让情感共鸣更直接 。**人物关联内容**：人像与笔记提及的人物（如余华相关）或情感主体契合，借形象强化对文字观点的诠释 。 |  |
| 种草 | **互动感强**：通过手势引导（如比耶、指向物品等）、表情传递兴奋 / 惊喜情绪，拉近与观众距离，突出 “分享” 核心。**场景生活化**：多在居家环境（客厅、卧室等）或简约背景中，营造轻松、日常的氛围，贴合 “好物融入生活” 的分享逻辑。**动作关联物品**：以手持、展示、使用物品为主要动作，直观呈现好物形态与功能，辅助文案传递价值。**风格亲和力**：妆容、穿搭偏向日常简约，神态自然亲切，强化 “真实用户分享” 的信任感，降低营销感。**多人出镜拓展**：部分笔记采用多人组合形式，通过不同视角展示好物，丰富内容层次，覆盖更多用户喜好。 |  |
