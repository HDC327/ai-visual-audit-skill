# 样机素材品质标准
### 设计原则及标准

| 设计原则 | 标准说明 | 正面案例 | 反面案例 |
| --- | --- | --- | --- |
| 元素完整 | 确保所有主体元素在画面中完整呈现，无缺失、断裂或大量遮挡情况。构图时裁剪需合理，避免影响视觉辨识或后续素材应用。 |  |  |
| 材质真实 | 精准还原样机所代表物品的真实材质，如金属反光、玻璃与塑料的通透感、布料的纹理与褶皱等，增强视觉可信度。 |  |  |
| 光影自然 | 通过合理的光源增强立体感，如侧光、柔光，避免过度打光或打光不足导致的失真 |  |  |
| 色彩协调 | 画面色彩搭配需遵循基本配色理论（如互补、类似色搭配），保持整体视觉的和谐统一。背景与主体需有色彩对比，突出样机主体。 |  |  |
| 造型准确 | 确保物品外形结构准确，比例与透视合理。在多角度展示中，造型需保持一致，避免视觉错位或变形。 |  |  |
| 构图平衡 | 画面布局应稳定、平衡，主次分明，符合视觉引导路径。避免构图失重、视觉倾斜等问题，提升专业感与可用性。 |  |  |
| 聚焦留白 | 画面保持简洁，适当留白，引导视觉聚焦于样机主体。留白区域避免杂乱内容干扰，确保主体处于视觉优先级的中心层级。 |  |  |
| *创意展示（非必须） | 在保持实用性的前提下，可结合创意视角、构图形式、背景设定等方式进行创新表达，增加吸引力与独特性，区别于常规展示样式。 |  | - |
| 易于延展 | 素材结构需图层清晰、命名规范，便于设计师快速替换贴图内容。保证修改贴图时不影响光影、材质等整体融合效果，提升工作效率。 |  | - |
| 画质清晰 | 输出需满足高清标准（建议300DPI），图像边缘清晰锐利。100%放大无模糊、锯齿、马赛克或杂色像素，适用于印刷与高清展示场景。 |  |  |

### 设计要点

| 样机素材分类 | 要点说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 包装盒/瓶/袋 | **强调结构与材质**：清晰呈现包装盒、袋、瓶等立体结构和材质质感（如纸张纹理、塑料光泽）。**真实比例**：保持与实际包装一致的比例，避免夸张变形。**多角度展示**：提供正面、侧面、俯视等角度，便于设计稿全面预览。**光影自然**：光影应贴合包装形状，增强立体感。**易于更换图案**：贴图区域清晰可编辑，适配多种设计稿。 |  |  |
| 电子产品 | **设备真实还原**：精准还原设备外观、比例与屏幕显示比例（如手机、电脑、平板）。**高分辨率屏幕表现**：屏幕部分应为高清晰度，突出展示内容。**干净现代的背景**：常用极简风格，突出产品本身。**反射与阴影真实**：反光和阴影模拟设备材质（如金属、玻璃）的真实效果。 |  |  |
| 服饰 | **自然穿着状态**：模拟衣物在人身上的自然垂坠与皱褶。**布料纹理清晰**：真实还原不同面料（棉、麻、牛仔等）的质感。**贴图位置准确**：确保图案贴图位置精准，不产生拉伸变形。**模特或无模特兼备**：提供带模特与平铺展示两种形式。**色彩还原真实**：避免滤镜过重，确保设计颜色准确预览。 |  |  |
| 办公/文具 | **细节清晰可见**：如笔记本纹理、纸张边缘、金属夹具等需清晰呈现。**排版友好**：素材预留充足贴图空间，方便放置LOGO、图形、文字。**实用组合展示**：常见组合如笔记本+笔、信纸+信封等，提升使用场景感。**角度简洁规范**：常用俯视或45°角，便于视觉整合。**辅助道具适度**：避免过多杂物干扰主体展示。 |  |  |
| 印刷物料 | **还原质感和效果**：展示不同纸类（哑光、铜版纸、牛皮纸等）质感、还原印刷工艺效果。**展示折页效果**：提供折叠、翻页、摊平等多种展示形态。**细节强化表现**：如折痕、光影、褶皱等能增强写实感的细节表现。 |  |  |
| 场景 | **真实环境融合**：广告牌应自然融入城市街道、商场、地铁等背景中。**透视关系合理**：保证广告画面符合透视规则，避免扭曲。**光照方向一致**：广告图与场景光线方向统一，增强真实感。 |  |  |
