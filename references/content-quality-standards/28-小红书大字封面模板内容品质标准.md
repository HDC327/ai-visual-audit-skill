# 小红书大字封面模板内容品质标准
该标准基于、，设计时需一并参考

| **设计原则** | **要点** | **说明** | **正面案例****​** | **负面案例****​** |  |  |
| --- | --- | --- | --- | --- | --- | --- |
| **和谐统一** | 文字 | **选择与主题和风格相匹配的字体，来传达情感或强化信息**单文本大字封面字体没有太多限制，多为较粗大的**标题类型字体和各种类型的手写字体，选择符合文案和整体视觉氛围的即可****若有副标题需**保证字体间搭配的统一性，通常来说：黑体配无衬线、宋体配衬线、圆体配圆体 |  |  |  |  |
| 图形图像 | **选择合适的图形图像如emoji、俏皮的卡通图案、****夸张的猫狗宠物表情、通用的喇叭等****，能够直观地传达主题**，图形和样式需要适应设计需求和整体视觉氛围 |  |  |  |  |  |
| 色彩 | 使用**不同配色文字**来区分文案，能够更好的强化信息和情感表达通过**理解色彩心理学的代表含义**，明确主题的情绪氛围，与主题保持统一 |  |  |  |  |  |
| 排版 | 使用版式重复原则来保证画面的整齐统一，可以是形式、字号、色彩、对齐方式的统一合理布局图文信息，考虑各元素之间的间距和比例，确保整体的视觉平衡感 |  |  |  |  |  |
| **层次分明** | 文字 | 通过**字号、字重、字色、投影、字效****、添加图形**等来区分标题、副标题和正文等信息的**层级结构** |  |  |  |  |
| 图形图像 | 使用**大小、颜色、位置、层级、虚实**来梳理图形的层次 |  |  |  |  |  |
| 色彩 | 通过**颜色的深浅或冷暖**来创建视觉层次，更好的引导观众的视线和注意力 |  |  |  |  |  |
| 排版 | 使用**版式亲密性原则**来梳理层级关系，并用**对比原则**来区分整体画面的层次把**文本与符合主题调性的 符号/色块/线条/文本背景 做结合**，增加画面层次结合**点线面**的处理，进行**动线引导** |  |  |  |  |  |
| **焦点强调** | 文字 | 通过**颜色、材质、字号、字重、字效、添加图形**等方式来**强调核心文案**信息 |  |  |  |  |
| 图形图像 | 使用符合需求的图形如**emoji**来强调主题文案 |  |  |  |  |  |
| 使用符合需求的图形如**卡通图案**来强调主题文案 |  |  |  |  |  |  |
| 使用**划重点**来强调主题文案 |  |  |  |  |  |  |
| 色彩 | 使用**强烈的对比配色**技法，来突出关键要传达的信息或视觉元素 |  |  |  |  |  |
| 使用不同配色的**划重点**来强调主题文案 |  |  |  |  |  |  |
| 排版 | 通过**强对比的排版原则**来强调焦点 |  |  |  |  |  |
| **传达清晰** | 文字 | **选择易读的字体**调整字号字重，间距，确保在不同的尺寸和距离下都能清晰阅读通过字色、字效、投影或立体效果来增加文字与背景的对比 |  |  |  |  |
| 图形图像 | 保证图像的合理性，**符合认知**图形图片的**主体完整**无残缺，整体造型结构协调使用**高质量、清晰度高的图片** |  |  |  |  |  |
| 背景 | **避免使用过度复杂/杂乱的图像**，以免分散观众的注意力注意**对比**，不影响前景文本或图像的识别 |  |  |  |  |  |
| **细节丰富** | 文字 | 使用填充（颜色/图案）、描边、错落、变形、材质等方式来丰富字体效果 |  |  |  |  |
| 图形图像 | 使用滤镜、描边、投影等特效来丰富图形效果效果 |  |  |  |  |  |
| 排版 | 通过给文本**添加点线面、色块、图形、符号等**元素来丰富细节巧用**相似或反差字体进行搭配**，使版式得到优化 |  |  |  |  |  |
| **创新创意** | 文字 | 关注当下字体的流行趋势，结合当下**流行的字体混搭或风格化字体**，来吸引用户阅读根据作品主题、受众对象的特征，通过**创意文案、创意字体**，来吸引受众对象的关注有**多材质、体积刻画**和细节处理等 |  |  |  |  |
| 图形图像 | 通过**同构、异构、解构、重构、拟人、矛盾、悖理、变异、多材质等创意手法**，创造出创意的画面 |  |  |  |  |  |
| 色彩 | 注意**色彩流行趋势**，尝试新颖的配色趋势，能够让海报脱引而出 |  |  |  |  |  |
| 背景 | 尝试**新颖的****滤镜**，让海报更具特色 |  |  |  |  |  |
| 排版 | 在遵循排版原则下，通过**反差、自由的排版方式**来吸引关注 |  |  |  |  |  |
| 通过**非常规的图框/拼图方式**使画面更加新颖 |  |  |  |  |  |  |
| **易于延展** | 文字 | **划重点、列表样式、文字背景等**元素需使用编辑器进行还原，保证编辑可用性 |  |  |  |  |
| 图形图像 | **图片容器、描边等**图像类型需使用编辑器进行还原，保证编辑可用性 |  |  |  |  |  |
| 色彩 | 矢量图形配色保持可编辑 |  |  |  |  |  |
| 背景 | 纯色或渐变背景配色保持可编辑 |  |  |  |  |  |
| 保持简洁，考虑内容的尺寸延展，**背景元素需尽量拆分** |  |  |  |  |  |  |
| 排版 | **合理分组**，同层级文案或临近元素打组 |  |  |  |  |  |
