# 图标素材内容品质标准
​

### 设计原则及标准

| **设计原则** | **标准说明** | **正面案例** | **反面案例** |
| --- | --- | --- | --- |
| 元素完整 | 确保主体元素完整，没有缺失的部分具备完整的功能表达，避免过度简化导致信息不清晰 |  |  |
| 造型清晰 | 图标造型能明确地表达要传递的功能或信息图标形态直观，造型清晰简洁不冗余，避免过多装饰或复杂造型叠加影响识别性 适当留白，避免图形过于密集影响清晰度 |  |  |
| 线条流畅 | 线条曲线平滑，避免锯齿感或不规则断点 转角处理柔和，可使用圆角过渡，使图标更精致 |  |  |
| 色彩协调 | 色彩搭配和谐，遵循色彩理论，以确保视觉效果的和谐统一色彩搭配符合行业调性渐变色使用自然、流畅的渐变过渡，避免生硬的颜色边界 |  |  |
| 视觉平衡 | 比例均衡，视觉平衡，重心居中，避免偏斜或倾斜感**形状大小协调**，确保同系列图标在视觉上均匀 |  |  |
| 材质细腻 | 如采用3D/拟物/轻质感类风格，材质应精细真实，增强视觉层次感加入**光影、渐变、纹理**等细节，增强图标质感 |  |  |
| 风格统一 | 同一系列图标保持风格一致，增强整体性（如统一的线条粗细、圆角半径、疏密关系、光影或肌理处理、透视角度等） |  |  |
| 画质清晰 | 保证高清画质，100%放大后图形边缘清晰锐化，无模糊感，无锯齿马赛克，无杂色像素。 |  |  |

### 设计要点

| **图标风格** | **要点说明** | **正面案例** | **反面案例** |
| --- | --- | --- | --- |
| 线性图标 | 线宽一致，避免粗细不均 圆角与直角风格统一，保证整体协调​ |  |  |
| 面性图标 | 抓住物体最具代表性的轮廓和形状，突出主题圆角与直角风格统一，保证整体协调 |  |  |
| 毛玻璃图标 | 通过高斯模糊等方式增强磨砂感 顶部可增加柔和的玻璃光感，同时可通过微调图标边缘厚度、叠加噪点、增加内反光等方式提升精致度 避免使用过于实、硬的描边，以免破坏柔和、朦胧的视觉感 |  |  |
| 拟物图标 | 通过渐变色、不透明度渐变、投影、内阴影等形成层次变化和光影效果，多图层刻画来强化图标质感和层次感 |  |  |
| 2.5D图标 | 透视角度和光源位置阴影方向需统一注意体积感的塑造，通过多层渐变叠加（高光、基础色、暗部渐变）和内阴影增加图标立体感投影自然，底部投影透明度一般为 20%-40% ，避免过深 |  |  |
| 插画图标 | 确定好透视角度（如平视视角），保持图标系列的视觉一致性 可以（根据需求）在色块中叠加一些轻微的颗粒、纸质或粗糙肌理，提升手绘质感 |  |  |
| 3D图标 | 整体光影、透视、结构与真实情况保持一致 根据图标的材质（如金属、塑料、玻璃）调整光泽、反射和肌理效果 边缘渲染平滑，避免出现锯齿或生硬的边线 |  |  |
