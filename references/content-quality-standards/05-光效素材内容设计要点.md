# 光效素材内容设计要点
| 材质特效 | 说明 | 正面案例 | 反面案例 |
| --- | --- | --- | --- |
| 光线 | 光线的线条需要流畅，不能有生硬断裂或不和谐的弯曲抖动；发光效果明显，强调光源或高光部分，最亮的部分颜色最实，最接近白色，距离高光或光源越远，光效和颜色越淡，呈现光的衰减和散射效果； |  |  |
| 光圈 | 由一条或多条光线组成的圆形或椭圆形光圈造型，光圈的线条要求流畅，符合圆形或椭圆形的曲率；如有透视角度，需绘制准确，近大远小，同时光线自身的亮度和粗细也应有规律地变化；发光效果明显，强调光源或高光部分，最亮的部分颜色最实，最接近白色，距离高光或光源越远，光效和颜色越淡，呈现光的衰减和散射效果； |  |  |
| 光束 | 表现光源放射出光束，光为锥状或放射状，由多条光线或多个光面组成；光的强度和边缘清晰度，离光源越近越清晰，离光源越远越模糊，直至渐变消失；光越强的位置，颜色越实，越接近白色； |  |  |
| 光晕/光斑 | 模拟透过镜头观察光源，呈现的光晕和虚化光斑效果；光晕为圆圈形状，它的圆心与光源应在同一直线上；需表现光晕的晕染效果；光斑需表现出层次感，不同大小、虚实、透明度的光斑随机错落分布；彩色光斑颜色渐变需符合光谱颜色顺序，即彩虹色顺序； |  |  |
| 光影 | 模拟光透过窗户或者植物缝隙，投到墙上的光影效果；窗户结构不宜太过复杂，以田字形结构为主；植物影子要有斑驳感，边缘模糊，造型优美不凌乱；越靠近光源的造型边缘越清晰，越远离光源的造型边缘越模糊；颜色以透明白色或灰色为主，白色绘制光亮，灰色绘制投影，光影素材单独绘制光亮或投影，二选一，不同时绘制； |  |  |
