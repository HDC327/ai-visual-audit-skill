# 新媒体套装设计要点
新媒体套装主要包含的物料类型为：手机海报、公众号首图、次图、横版海报、公众号首次图

| **案例** |  | **要点** |  |  |
| --- | --- | --- | --- | --- |
| 手机海报 |  |  |  | 包含促销活动、优惠价格、详细规则等重要信息的长文案内容，**避免使用竖排版**，以免影响用户编辑体验及长文字内容的阅读性 |
| 首图 |  |  |  | 文字排版清晰（明确左中右排版）；标题位置不可过高或过低；标题文字切忌过大logo、二维码去掉 |
| 次图 |  |  |  | 主要展示文字为主；文字字号统一元素避免过多，挤占文字空间 |
| 横版海报 |  |  |  | 主要展示文字为主；文字字号统一元素避免过多，挤占文字空间 |
