# 贴纸工具素材品质标准
### 设计原则及标准

| 设计原则 | 标准说明 | 正面图例 | 反面图例 |  |
| --- | --- | --- | --- | --- |
| 元素完整 | 主体完整性 | 确保主体元素完整，避免关键部位缺失或出现多余部分 |  |  |
| 裁切合理 | 合理的裁切，使元素保留元素应用空间，不影响素材实际使用​ |  |  |  |
| 搭配和谐​ | 氛围一致 | 配色、材质与风格、氛围的调性一致，给人舒服的视觉感受举例：潮流、科技、清新 |  |  |
| 色彩协调 | 色彩搭配和谐，遵循色彩理论，以确保视觉效果的和谐统一 |  |  |  |
| 内容关联 | 贴纸里的元素与元素之间需要有合理的互动的关联性，避免出现互相独立或无关联性的存在 |  |  |  |
| 辨识度高​ | 含义清晰 | 元素含义不会导致歧义，有较好的识别性以及清晰的信息表达举例：花、情侣、骂人 |  |  |
| 结构准确 | 结构准确，符合物理或风格化条件，避免复杂的造型结构 |  |  |  |
| 结构流畅 | 元素线条、边缘走势曲线流畅合理，转折处圆润处理，避免出现生硬或断裂的线条和崎岖的边缘 |  |  |  |
| 美观实用​ | 造型简约 | 避免元素内外带有过多细节和氛围装饰，以实用为主 |  |  |
| 材质细腻 | 准确细腻的的材质表现，选择适当合理的纹路表现材质质感 |  |  |  |
| 明亮干净 | 除特定风格求外，通常需保证元素整体干净明亮，避免灰暗、肮脏或杂乱 |  |  |  |
| 画质清晰 | 保证高清画质，100%放大后图形边缘清晰锐化，无模糊感、无锯齿马赛克、无杂色像素，材质 |  |  |  |
| 背景处理 | 元素必须完全去除背景（像剪纸图案一样干净）检查边缘和空白部分不要残留白边、黑边、杂边、杂点或投影 |  |  |  |
| 分层改色 | 多个元素组合的贴纸需在源文件中将元素与元素的分层，确保可以对元素进行拆解或改色，具体参照执行 |  |  |  |

### 设计要点说明

素材的品质除了符合设计原则及标准外，设计时需确保符合以下设计要点

#### 套系要点

| 设计要点 | 标准说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 风格统一 | 同套系内的贴纸需要的设计风格保持一致，如果为同一角色需要保证头身比和角色特征也统一 |  |  |
| 色感统一 | 同套系内的色彩、饱和度、明度应具有关联性，色彩感官一致 |  |  |
| 细节统一 | 同套系光影条件、材质纹理质感、线条粗细等都应保持一致，如有透视角度要求也需要统一 |  |  |

#### 生产类型要点

##### 照片贴纸

| 设计要点 | 标准说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 边缘干净 | 抠图边缘干净顺滑，没有残留白边或者其他内容 |  |  |
| 裁切合理 | 元素尽可能保持完整，或者有较合理的裁切方式如花朵、数枝保留一定的枝干，人物露出手部脚部时需保持完整​ |  |  |
| 无遮挡物 | 元素干净，前景无模糊、凌乱以及其他影响元素呈现的遮挡物 |  |  |
| 文字清晰 | 元素上的文字内容均清晰且合理，如有 logo 需去除 |  |  |

##### 拼贴贴纸

| 设计要点 | 标准说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 裁剪合理 | 边缘像用剪刀裁剪过，需要干净利落，没有过于细碎的边角或背景画面 |  |  |

##### 3D贴纸

| 设计要点 | 标准说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 质感明确 | 元素塑造符合要表现的质感，材质的光影和纹理清晰表现符合所需质感举例：粘土、玻璃、针织面料 |  |  |
| 光影干净 | 布光干净，避免出现杂乱、发灰、暗面过强的情况 |  |  |
| 元素完整 | 3D 类元素需要完整闭合的轮廓，不要直接裁切元素边缘 |  |  |
| 面数合理 | 除lowpoly外，避免出现低面数导致的不平滑的情况 |  |  |
| 衔接合理 | 元素各部位衔接合理，避免出现明显的不平滑、破面、错位、穿模等情况 |  |  |

##### 矢量贴纸

| 设计要点 | 标准说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 曲线流畅 | 边缘曲线流畅，避免元素转交生硬 |  |  |

##### 手绘贴纸

| 设计要点 | 标准说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 笔势流畅 | 笔势流畅，粗细变化合理，外轮廓线＞细节线，前景线＞背景线，避免不合理用线 |  |  |
| 填色均匀 | 颜色填充合理，避免填色不均匀或发脏 |  |  |

##### AI 贴纸

| 设计要点 | 标准说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 结构合理 | 元素的细节符合对应的结构逻辑，尤其是对于手、脚、丝带走势等细节需要更加注意，错误的内容需要进行修复 |  |  |
| 元素干净 | 元素周围保持干净，避免多余或不合理的装饰，多余的元素需修复，确保素材整体干净 |  |  |
| 风格准确 | 生成的元素的材质、质感、纹理与风格需求匹配举例：毛毡、3d、镭射金属 |  |  |

#### 元素要点

若涉及到以下内容生产时还需注意符合对应的元素设计要点

##### 人物元素

| 设计要点 | 标准说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 特征明显 | 角色的年龄、性别、职业、关系明显，避免出现特征不清的情况举例：儿童、女孩、母亲 |  |  |
| 构图合理 | 除特殊需求，通常人物比例需合理，且根据人物动态进行合理构图，手足末端、袖子、衣摆、头发尽可能完整露出 |  |  |
| 裁切合理 | 裁切位置应尽量避开人体关节处，尽量呈现较完整的内容。头像可仅有完整的头部包含发型，胸像则需在头顶至上臂1/2处，半身需到达臀部左右，而全身则需要完整的身体 |  |  |
| 关节需求 | 除风格化需求外，带有动作相关的内容，通常需要保留部分关节部分，如有袖子，则需保留袖口 |  |  |

##### 动物元素

| 设计要点 | 标准说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 特征明显 | 符合对应动物具体的特征和细节，防止物种混淆举例：兔子、蛇、水獭、东方龙 |  |  |
| 碎片识别性 | 碎片化呈现仍需具备品种识别性，如兔子朵长且圆，避免处理成尖儿等举例：兔耳、狗鼻子、虎爪 |  |  |

##### 自然元素

| 设计要点 | 标准说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 细节准确 | 植物的细节确保准确，根与叶、花的穿插正确，确保植物本身的完整性，同时对不需要的部分进行删减 |  |  |
| 衔接干净 | 元素衔接需要注意不要出现不合理的透层或交叉情况，需要清理干净确保元素展示合理 |  |  |

##### 艺术字

| 设计要点 | 标准说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 辨识度高 | 避免字形过于花哨、笔画不清晰、线条过细、过粗等导致文字识别困难的情况 |  |  |
| 字效干净 | 材质/肌理/配色需运用得当，避免因字效设计影响文字的阅读性 |  |  |
