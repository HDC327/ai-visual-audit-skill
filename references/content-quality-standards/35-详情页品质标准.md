# 详情页品质标准
该设计要点基于[版权内容品质标准V1.0-2412-海报模板](https://doc.huanleguang.com/pages/createpage.action?spaceKey=~guojiang&title=%E7%89%88%E6%9D%83%E5%86%85%E5%AE%B9%E5%93%81%E8%B4%A8%E6%A0%87%E5%87%86V1.0-2412-%E6%B5%B7%E6%8A%A5%E6%A8%A1%E6%9D%BF&linkCreation=true&fromPageId=439347955)，设计时需一并参考

| 设计原则 | 设计要点 | 说明 | 正面案例 | 负面案例 |
| --- | --- | --- | --- | --- |
| 和谐统一 | 了解目标受众 | 分析目标用户的年龄层次、性别、消费习惯、兴趣爱好等。针对年轻时尚的消费群体，设计风格可偏向潮流、活力；面向商务人士，则注重简约、专业。若产品是高端护肤品，目标受众可能是有一定消费能力、注重肌肤保养的女性，设计需营造出优雅、奢华的氛围。 |  |  |
| 色彩搭配统一 | 详情页的整体视觉风格应与品牌形象保持一致，包括色彩搭配、字体选择、图标设计等，营造出专业、协调的购物氛围。例如，对于香水产品，可以运用淡雅的色彩和精致的排版，来展现出产品的优雅气质。对于3C数码类产品，通常会采用更加现代、科技感的颜色和设计元素。 |  |  |  |
| 场景化展示 | 模拟产品的实际使用场景，帮助用户想象产品如何融入日常生活，产生情感共鸣。 |  |  |  |
| 层次分明 | 逻辑动线 | 通过将详情页内容划分为不同的模块，如商品核心卖点、使用场景、细节展示、用户评价、售后保障等。使页面结构清晰，用户可以快速找到所需信息。 |  | / |
| 视觉区分 | 利用不同的排版方式（如左右、上下、拼图或满版排版）和视觉元素（如分隔线、留白、颜色对比等），在视觉上区分不同内容板块，增强层次感。**左右排版**：适合对比展示产品的不同特点或使用场景，通过左右对称的布局，清晰呈现对比信息。**上下排版**：适合按顺序展示产品的多个角度或使用步骤，通过上下排列，引导用户逐层了解产品。**拼图排版**：适合展示产品的多个细节或使用场景，通过拼图式的布局，让用户全面了解产品。**满版排版**：适合突出产品的整体形象或核心卖点，通过满版的图片或设计，吸引用户注意力。 |  |  |  |
| 信息优先级 | 通过调整各模块的大小、位置和设计，体现信息的优先级，使重要信息更加突出，次要信息作为补充。将最重要的卖点放在最显眼的位置（如顶部），次要卖点依次排列，确保用户第一时间抓住关键信息。 |  |  |  |
| 焦点强调 | 首屏大图 | **首屏吸引：**使用高质量的主图或创意海报在首屏展示，迅速吸引用户眼球，3秒内传达产品的核心卖点。 |  |  |
| 卖点罗列 | 利用放大字体、醒目颜色或使用搭配形象、表意明确的图标、图形或插图等设计手段辅助，突出展示商品的核心卖点，使其在众多信息中脱颖而出。 |  |  |  |
| 局部放大 | 提供图片放大功能，方便用户查看商品细节，增强用户对商品的信任感。 |  |  |  |
| 传达清晰 | 使用高清的商品图 | 使用高清、清晰、具有吸引力的商品图片，从不同角度展示商品的外观、细节和使用场景，让用户能够直观地感受商品的特点和品质。 |  |  |
| 优势对比 | 通过与竞品或使用前后的效果对比，突出产品的优势和价值，增强用户的购买意愿。 |  |  |  |
| 使用可视化图形 | 通过图形化的方式展示产品的核心功能、优势或使用场景，让用户快速了解产品的价值 |  |  |  |
| 产品相关的数据或性能指标，可以使用图表、图形等形式进行可视化展示，使信息更直观易懂如：对于产品的性能数据、用户评价数据等，可以用柱状图、折线图、饼图等形式展示。 |  | / |  |  |
| 行动号召 | 添加立即购买/收藏/咨询客服等按钮设计 |  | / |  |
| 参数罗列 | 采用表格或列表的形式展示参数，使信息更加清晰。表格适用于参数较多且需要对比的情况，列表则更简洁。<br>依据属性分类：电子产品分性能、外观、规格等服饰分尺码、材质、颜色等 |  | / |  |
| 建立信任 | 展示品牌资质、用户评价、售后服务等信息，增加消费者对商品和品牌的信任度，减少购买疑虑。 |  | / |  |
| 简洁明了 | 文字简洁明了 | 首屏文字信息应简洁，突出产品名称、核心卖点或品牌价值深入分析产品的独特优势和价值，包括品质、功能、设计和价格等方面，提炼成简洁有力的卖点文案。 |  |  |
| 色彩简洁明快 | 使用简洁的背景（如纯色、浅色或渐变色）来增强主体的视觉效果，避免复杂的背景分散用户注意力。如果需要展示使用场景，背景应与产品风格一致，且不过于复杂。 |  |  |  |
| 留白得当 | 元素布局协调 | 在信息模块之间留出足够的空白区域，避免页面过于拥挤，提升整体的视觉舒适度。留白可以引导用户的注意力集中在重要信息上。 |  |  |
| 创新创意 | 视觉吸引 | 采用独特的色彩搭配、新颖的排版布局或富有创意的视觉元素，画面风格根据当下流行趋势，区别于常规平铺展示 |  | / |
| 细节丰富 | 产品光效细节 | 利用光影效果增加画面的真实感和质感，使产品看起来更加赏心悦目，提升档次 |  | / |
| 易于延展 | 图片要求 | 单张图片高度不超过2000px，宽度在480px-1600px间，大小不得超过3M，否则可能会提示异常。为获得更好展示效果，建议您在做图时，单图宽度设为620px，高度设为不超过960px。高度超过960px的部分图片可能会触发平台的切图操作，因此会改变图片的实际尺寸。此改变不会影响您图片的正常展示，请您无需担心。图片大小≤3M，支持jpg、jpeg、png格式 | / | / |
| 文字要求 | 文字字数≤25000字 | / | / |  |
| 内容长度限制 | 宝贝详情装修**总高度限制调整为100,000px**（25年2月），可通过页面右下角【剩余可装修高度】进行查看。 |  | / |  |
| 表格样式设计 | 详情页里的产品信息模块，先在设计稿里进行设计，审稿通过后，到编辑器上传时需用表格组件进行还原。在设计时，以通用样式为主，避免无法还原的情况。 |  | / |  |
