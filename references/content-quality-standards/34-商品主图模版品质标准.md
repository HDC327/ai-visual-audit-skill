# 商品主图模版品质标准
- 该设计要点基于[版权内容品质标准V1.0-2412-海报模板](https://doc.huanleguang.com/pages/viewpage.action?pageId=425001173)，设计时需一并参考

### 基础视觉规范

| **设计原则** | **设计要点** | **说明** | **正面案例** | **负面案例** |
| --- | --- | --- | --- | --- |
| 和谐统一 | 文字统一 | 字体符合**商品的定位，**需要与**品牌的定位**和**目标市场相**吻合。不同的字体风格传达不同的品牌性格，选择时要确保字体能够反映品牌的核心价值。例如：现代简约字体适合科技、创新型公司，传递出简洁、前沿的感觉，而经典字体则适合奢侈品、金融公司，传递出稳定性、可靠性与高端感轻奢：衬线字体+极细线条科技：非衬线字体母婴：圆润手写体（避免全大写字母）运动户外：粗无衬线体+斜体图书文具：手写体+传统宋体宠物用品：圆润字体+趣味图标字体 |  |  |
| 布局统一 | 保持布局的一致性，如**对齐方式**、**元素间距**等，使整体看起来更加和谐。 |  |  |  |
| 颜色统一 | 选择与**产品颜色相近**的同色系背景色以营造和谐感（但要保证产品的突出，产品不能与背景融于一体）对于黑白灰产品可以纯色调或者简单背景以增强视觉吸引力，同时确保背景色与产品色不冲突，以维持整体视觉的平衡 |  |  |  |
| 背景关联性 | 背景与产品的用途和**特性相关联；****例如：****家居产品：**一款新的沙发可以在一个现代客厅的背景中展示，以突出其设计感和舒适度。**户外装备**：一款露营帐篷的主图可以将其放置在森林或山地的背景中，展示帐篷搭建在自然环境中的样子，以此突出产品的户外使用场景。这种展示方式能够**让消费者直观地感受到产品如何融入他们的家居生活，**从而提高购买意愿。 |  |  |  |
| 焦点强调 | 突出主体 | **清晰展示商品：**主图的核心是商品本身，必须确保商品是画面的焦点图片在画面中占比：50%-60%留白空间：20%-40% |  |  |
|  | **突出卖点/促销点：**可以通过**腰带图、图标、标签、**等方式来强调商品卖点，但要注意不要过多或过杂，影响主图的美观和清晰度。 | 【腰带图】 | ​ |  |
|  |  | 【标签】 | ​ |  |
|  |  | 【图标】 | ​ |  |
| 传达清晰 | 易读性 | 使用**清晰易读**的字体，确保文字信息在各种屏幕尺寸下都能被轻松阅读 |  |  |
| 图片处理 | 使用图像处理软件对商品进行修饰，避免**过度曝光**或**投影过重** |  |  |  |
| 背景 | 背景颜色应与商品颜色**形成对比**，以增强商品的可见性。 |  |  |  |
|  | 优先使用**高质量、高分辨率**的图片或者图形，确保画面整体精致度 |  |  |  |
| 简洁明了 | 视觉简化 | 避免过多元素干扰：尽量**减少与商品无关**的背景或装饰元素，以免分散用户的注意力 |  |  |
| 留白得当 | 位置合理 | 文字应放置在画面的**空白区域**，**避免遮挡商品主体** |  |  |
| 空间平衡 | 通过留白**突出商品**和**其他重要信息**，避免视觉上的拥挤 |  |  |  |

### 易用性标准：

| **设计原则** | **设计要点** | **说明** | **正面案例** | **负面案例** |
| --- | --- | --- | --- | --- |
| 易于延展​ |  | **矢量图形保留可编辑**色块/icon/图形/图框等尽量要保持可编辑，目前支持多种模式下的矢量图形改色（**支持填充&描边纯色和渐变色**）注意：凡是纯色的图形图层，必须用形状工具还原成矢量，便于用户进行编辑改色**​**形状图层**不能添图层样式**效果，无法识别；如需做投影效果，还是栅格化成图片来做，上面再新建一个无图层样式的形状色块，便于用户修改颜色 |  |  |
| ​ |  | **抠图分层：**当设计为**抠图样式**时，**产品和背景必须分层处理**，用户才能将产品替换成自己的。不要直接找一张好看的图，产品和背景无法分层，用户无法替换还原场景效果；合理保留分层，最大限度确保用户可编辑，可还原视觉，避免造成替换效果差异过大。 | 图框样式：有明显边框或显示区域限制抠图样式： |  |
|  |  | 图框效果不可直接与图框合并整理文件需要保证图框效果（描边、投影等），需要将图框效果层独立出来，否则用户在替换产品同时会连同效果一起替换。图片效果若是能在编辑器里还原的需直接在编辑器还原 |  |  |

### 补充：

**不同商品品类在主图设计上各具特色，需要根据其特定属性和目标受众的偏好来量身定制。**

| **要点** | **图例** |
| --- | --- |
| **电子产品**： 电子产品的主图通常需要突显关键技术和产品作用。这种产品的用户在选择时通常更重视产品的性能和参数。主图应清晰显示产品的整体外观，加上一些特写，重视核心技术。例如耳机的主图可以包括耳机接口、音量调节按键等特写，帮助消费者更好地产品的功效。环境要简单，确保商品本身变成视觉焦点。 |  |
| **时尚产品**： 在时尚产品中，如服装和饰品，主图的风格更重视视觉美和搭配效果。本产品的用户关注趋势和企业形象，因而主图通常需要运用模特来宣传和展示服饰效果。服饰产品主地图应选用高质量相片，以保证真实色调，并呈现细节。 |  |
| **家居配件**： 针对家居配件，对主图的需要更偏重于展现好用的设计感。家居产品通常需要在照片中展现适用情景，以帮助消费者产品功能和适用性。 |  |
| **健康和宠物、美妆产品**： 在健康和宠物产品中，主图的需要更重视成份效果的展示。消费者在选择这类产品时，通常会关注成份的安全性和有效性。主图可以包含产品的详细信息，如成分使用体验。整洁简约的环境有益于提高产品的专业性和信任感，吸引消费者的购买欲望。美妆还有一个是赠品信息 |  |
| **食品和饮料产品**： 食品和饮料产品的主图必须突显产品的新鲜度和味道。通常对食品的外观和包装设计有较高的要求主图应尽量展现产品的实际色彩和纹路。 |  |
| **机械设备和工具**： 核心价值在于 “功能性、可靠性与易用性”；主图应关注产品的关键作用和使用方法 |  |

*以上图例部分为站外案例，仅做展示使用
