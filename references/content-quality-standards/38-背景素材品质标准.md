# 背景素材品质标准
### 设计原则及标准

| **设计原则** | 标准 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 色彩协调 | 色彩**搭配和谐**，遵循色彩理论，如互补色、相近色等，使视觉感受舒适。多色渐变需**过渡自然柔和**，有层次感。 |  |  |
| 构图平衡 | 构图保持**视觉平衡**，无论是对称还是非对称，都应给人**稳定和有序**的感觉。通过元素**重复、对比、对齐**等手法，达到画面视觉重量平衡。 |  |  |
| 线条流畅 | 画面中的光线、几何线条、元素线条，都需要**流畅有张力**。 |  |  |
| 质感明显 | 材质纹理表现需要符合该**材质特性**，画面美观，具体不同材质纹理表现要求**详见下方“设计要点说明”表格**。 |  |  |
| 简约留白 | 避免元素拥挤，尽量**精简元素**。平铺纹理需要**降低纹理存在感**。**色调简约**统一，不宜添加太多对比颜色。 |  |  |
| 突出主体 | 使背景图在应用时添加主体元素后，主体元素足够突出。通过线条、光影等视觉引导，使画面有**聚焦点**。画面**预留**足够的**空间**。**精简视觉层级，**去除多余干扰。 |  |  |
| 有空间感 | 通过画面元素前后层次的组合排列、色彩对比、光影虚实变化等手法，增加画面**纵深维度**的表现，营造空间感。 |  |  |
| 营造氛围 | 通过适当的**装饰元素**来表达画面主题氛围。有细腻的**光影效果**，精准的**材质表现**，让画面有精致的质感氛围。利用合理的**色彩搭配**表达画面情绪氛围。 |  |  |
| 画质清晰 | 保证高清画质，100%放大后图形边缘**清晰锐化**，无模糊感，无锯齿马赛克，无杂色像素。 |  |  |
| 易于延展 | 考虑用户体验，提供最大程度的**可编辑性**，如源文件分层细致合理，元素保持完整等，以便素材延展和适配不同场景。 | - | - |

### 设计要点说明

| 要点 | 说明 | 正面图例 | 反面图例 |  |
| --- | --- | --- | --- | --- |
| 光影效果 | 光线 | 光线线条流畅且发光效果明显；表现出光线流动或发光反射等特点；多条光线的时候要有明暗主次关系。 |  |  |
| 投影 | 真实的光影效果，光透过窗户或者缝隙投到墙上或地面，边缘过渡柔和，光角度统一；离光源越远的地方，投影形状和边缘越模糊。 |  |  |  |
| 光点光斑 | 点光源透过镜头或者玻璃，产生的虚化效果的光点光斑，随机分布，错落有致，单色或多色，营造浪漫的氛围感。 |  |  |  |
| 材质 | 金属 | 突出金属光泽感，明暗对比强烈，要有高光的刻画和暗部细节；表面呈现抛光或拉丝纹理，纹路细密有规律。 |  |  |
| 玻璃 | 表现玻璃材质的通透感，区分透明玻璃和磨砂玻璃；玻璃需要有一定厚度，注重边缘结构的光影刻画；光在玻璃上的反射和折射效果需还原。 |  |  |  |
| 流体 | 表现流体的动态和流动的特质，画面有动态感；丰富细腻的的色彩渐变，画面具有层次感；流体的动态曲线需符合流体力学，流畅自然，有较为统一的流动趋势。 |  |  |  |
| 水墨水彩 | 有清晰的笔刷纹理，能分辨得出笔刷走势和造型；有流畅的线条和优美的造型；有明显的笔触效果或水墨水彩的晕染效果；笔刷颜色与整体背景融合。 |  |  |  |
| 丝绸 | 表现丝绸柔软光滑的质感；需要有优美的动态曲线和自然的褶皱起伏，褶皱不宜过多，丝绸整体应是平滑的；刻画高光阴影，需有明显丝绸布料纹路； |  |  |  |
| 纸质 | 能清晰分辨出是纸张表面纹理，纹理存在感不需太强；颜色为纸张自身颜色或者叠加以单色，部分纸质表面会有轻微折痕和褶皱痕迹； |  |  |  |
| 木纹 | 能清晰分辨出是木纹表面纹理，纹理存在感不需太强；颜色为木质自身颜色或者叠加以单色，部分木纹表面会有年轮或扭曲线条。 |  |  |  |
| 水纹 | 表现水面真实水波纹涟漪纹理，有动态感，表面有反射高光；需要有通透感，颜色以蓝色绿色及透明白色为主。 |  |  |  |
| 大理石 | 表现大理石表面光滑细腻的质感，表面需要有天然大理石的纹路线条，线条走向需要有较为一致的走向，一般为倾斜走向；纹理存在感不需太强，颜色为大理石自身颜色或者叠加以单色。 |  |  |  |
| 粒子噪点 | 由粒子均匀或有规律地分布，组成规律或抽象的造型；粒子以噪点的形式均匀分布平铺于画面中，颜色与整体背景融合；粒子需清晰且颗粒细密，有层次感； |  |  |  |
| 平铺纹样 | 具有丰富细腻的纹样，线条流畅，有规律；若无规律图案则需繁复且有一定线条艺术抽象化处理；常见平铺纹样：鱼鳞纹、海浪纹、祥云纹、福字纹、线描图纹样等。 |  |  |  |
| 抽象几何 | 线条和形状分布具有几何规律美感，如对称、等距、等差排列、S曲线、抛物线、黄金分割等；使用线条或色块构成，整体画面视觉平衡有序，有空间感和层次感； |  |  |  |
