# 应援模板品质标准
## 一、设计要点

该设计要点基于，设计时需一并参考

### 基础视觉标准

| 设计原则 | 设计要点 | 说明 | 正面案例 | 负面案例 |
| --- | --- | --- | --- | --- |
| 和谐统一 | 符合受众调性 | **明确目标受众**的年龄、性别、兴趣爱好等，根据目标受众的偏好选择视觉元素、色彩和语言风格。比如深色类型一般选择舞台装扮的人物；清新甜美可爱等明亮的类型选择男大女大造型装扮的人物。 |  |  |
| 色彩契合 | **色彩选择要契合氛围**，从需求配色选择人物，或从人物配色延展整体配色。 |  |  |  |
| 层次分明 | 空间位置引导 | **视觉路径引导**：利用线条、箭头、人物视线或其他引导元素，将观众的注意力自然地引向焦点。 |  |  |
| 排版层次合理 | 合理**划分海报区域**，将标题、人物图片等内容有序排放。常见布局三分法有 “上下结构”“左右结构”“左中右结构”“中心聚焦/全包围/半包围结构” 等，确保信息层次分明。 |  |  |  |
| 焦点强调 | 突出人物 | **黄金比例法则：**将**主人物**位于画面的核心区域，对小图框或小的抠图人物进行陈列布局；通过**大小对比、色彩对比等方式来突出主人物。** |  |  |
| 突出标题 | **重复与强化**：对于重要的信息，可以通过适当重复或在不同位置进行强调，增强观众的记忆效果。以**较大字号、独特颜色、形状或特效突出呈现**。 |  |  |  |
| 传达清晰 | 文字清晰易读 | 注意**字体选择，间距保持合理**，保证各层级信息的可读性。 |  |  |
| 图像清晰易懂 | 使用高质量、高分辨率的图像，避免模糊或失真的图片。图像应与主题紧密相关，能够直观地传达产品或活动的核心内容。 |  |  |  |
| 留白得当 | 元素布局协调 | 各**元素间保持适当间距**，避免过于拥挤或松散，使海报整体布局平衡、协调。 |  |  |

### 优质视觉标准

在基础视觉标准之上，增加要求。

| 设计原则 | 设计要点 | 说明 | 正面案例 |
| --- | --- | --- | --- |
| 创新创意 | 文字类创意 | 主题类文字可用**新颖的材质/风格**表现，如玻璃浮雕、发光、酸性、肌理感、云朵外形、涂鸦手写字、描边拼贴感等字体效果，来增加视觉氛围。根据作品主题、人物特征，**巧用相似或反差字体进行混搭**。通用可不可编辑文字：happy birthday |  |
| 图像类创意 | 人物图片通过特殊效果处理来彰显创意，如添加网点效果、渐变映射、光感效果等，或给人物添加卡通耳朵、腮红等装饰丰富细节，使海报在众多竞品中脱颖而出。 |  |  |
