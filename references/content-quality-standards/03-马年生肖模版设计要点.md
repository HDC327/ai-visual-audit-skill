## 2.抽象/具象生肖设计要点

### （1）具象类生肖

#### 卡通

| **特征类型** | **正面案例** | **反面案例** |  |  |
| --- | --- | --- | --- | --- |
| 鬃毛 | 沿颈部自然生长 |  | 没有鬃毛 |  |
| 马颈 | 脖子偏短粗 |  | 没有脖子 |  |
| 马蹄 | 4条腿都必须呈倒梯形（ 可以是偏圆润的梯形） |  | 没有马蹄 |  |
| 脸部色块 | 明显的色块区分 |  | 没有色块 |  |
| 脸部形状 | 长脸形正面：呈圆润的矩形 / 长椭圆形，纵向拉长的鸡蛋侧面：嘴部比例缩短（约占脸长 1/2 ），流畅的 "S 形曲线" |  | 脸过圆 |  |
| 耳朵 | 马耳短而尖 |  | 耳朵过于短圆耳朵过于长，像驴 |  |
| 鼻子 | 小圆点或短弧线，与嘴部保持 1 个眼睛宽度的间距 |  | 鼻子过于宽，像河马 |  |

### （2）抽象类生肖

#### 卡通

| **要点** | **类型** | **特征** | **正面案例** | **负面案例** |
| --- | --- | --- | --- | --- |
| **核心特点**：用极度简化的符号代表马，几乎抛弃真实结构，只保留 “约定俗成的马的标志”，风格偏向趣味化、符号化​<br>​<br>​<br>​ | 几何抽象 | **造型简化**：把马的身体、四肢、头部等结构，用圆形、方形、三角形等基础几何形状拼接。**线条 / 色彩概括**：用简洁的线条勾勒轮廓，配色大胆鲜明，常搭配波点、条纹等几何图案装饰。**元素夸张变形**：打破现实中马的比例，比如头大身小、四肢缩短，或加入拟人化细节（比如大眼睛、装饰性花纹），整体可爱且具有装饰感。 |  |  |
| 字形抽象 | 把马的形态（比如马头、马身、四肢）融入文字的笔画结构里，让文字和马的形象相互嵌套、融为一体 |  |  |  |

#### 写实

| **要点** | **类型** | **特征** | **正面案例** | **负面案例** |
| --- | --- | --- | --- | --- |
| **核心特点**：以真实马的局部特征为素材，通过解构、重组、变形来表达主题，而非呈现完整的马​<br>​<br>​<br>​ | 意象抽象 | 常用水墨晕染、笔触飞白、色彩渐变等艺术手法，简化马的肌肉、毛发等细节，用抽象的色块、线条（比如飘逸的鬃毛简化为流动的曲线）塑造形象 |  |  |
| 几何抽象​ | 把马的身体、四肢、头部拆解为三角形、方形、圆形等几何块面拼接（比如马身用色块分割、四肢简化为线条 / 几何柱体），部分加入几何纹样（如波点、条纹）装饰，以平面化、装饰性为主，弱化写实的肌肉、光影细节，通过块面组合体现马的动态 |  |  |  |
| 字形抽象​ | 把马的关键形态（如马头、鬃毛、四肢）融入文字的笔画结构中 |  |  |  |

## 3.禁止or避免设计类型

| 说明 | 图例 |
| --- | --- |
| 禁用 “拉磨马、跛脚马、瘦马残鞍” 等造型暗示运势不佳：这类造型易让人联想到不顺、困顿、衰退等负面联想，与马年祈福纳祥的核心诉求相悖 |  |
| 禁用贬义词，如:马后炮 |  |
| 避免像驴 |  |
| 避免马身加入其他动物特征元素，如鹿角 |  |
| 不建议使用斑马造型动物分类: 斑马虽然在英文中也叫“zebra”，但它是非洲的一种野生马科动物。在中国古代，人们主要接触和饲养的是家马(马)，因此家马被选入十二生肖。 |  |
| ​不建议使用海马造型海马属于脊索动物门、鱼纲、刺鱼目、海龙科、海马属，是鱼类的一种。 |  |
| 避免**红眼睛（显凶狠 / 怪异，破坏亲和力）** |  |
| 忌冲克生肖同框<br>生肖马与鼠相冲，中式传统画风里避免同时出现两者生肖 |  |
| 避免 **融入西方独角兽元素（角、翅膀的奇幻设定）** |  |
