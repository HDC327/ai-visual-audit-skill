# AIGC元素设计要点
## 一.质量标准

### A.正确性

您在AIGC成图中，若遇到一些问题，需先对图像进行修复或自审，方可进行上传审核。

| **分类说明** | **具体标准** | **错误案例** |
| --- | --- | --- |
| 人体 | **人体如四肢，五官面部符合常规认知**不可出现肢体残缺、关节错位、面部崩坏、比例失调等情况不可出现同真实人物、知名游戏、动漫IP人物相同，相似，神似等情况 |  |
| 动物 | **动植物造型符合常规认知，能清晰识别品种**写实类动物不可出现肢体增生、品种组合、比例异常等情况；植物不可出现品种嫁接，花/叶/果不同种类 |  |
| 物品 | **物品造型符合常规认知，能清晰识别物品类型**几何形体物品线条流畅，不可出现细节变形。方/圆造型规整 |  |
| 建筑 | **建筑结构清晰，直线规整，曲线流畅**能清晰识别建筑廊柱、屋檐等内容，非模糊效果设计手法的画面需能清晰识别建筑主体及轮廓 |  |
| 光效 | **线性光效需具备流畅度与连续性，点状光效分布均衡，造型完整**不可出现光效元素不一致，线元素堆叠、杂乱、断开等情况 |  |

### B.版权安全

创作者提交审核前，也需对内容中元素细节进行检查，以避免出现AIGC成图中以下常见问题：

| **分类说明** | **具体标准** | **需求案例** | **正确案例** | **错误案例** |
| --- | --- | --- | --- | --- |
| IP | **不能生成现有所有带版权的ip形象***如：米老鼠，白雪公主，京东狗等等**不能生成明显的大ip中的拆分元素***如：雪宝的头，艾莎的衣服，米老鼠的手套等 | 迪斯尼风格的老鼠IP |  |  |
| 产品 | **不能生成特征较强的产品***如：茅台酒瓶，可口可乐瓶身，iphone“刘海屏”等具有较强识别度的产品造型 | 可乐手机​ |  |  |
| 人物 | **尽量避免直接使用真人照片进行AI垫图，特别是名人肖像，关键词不加人名，禁止生成明星脸；如需使用真人照片垫图，需与垫图做出较大区分，避免版权问题，不过****为避开肖像雷同尽量使用-文生图****☆重要：****如使用真人lora，为避免版权和雷同问题，需多个结合，不要只使用一个** | 一个年轻漂亮的女性​ | 一个美女√ | 一个长得像（xxx）的美女 |
| 图形 | **不能生成带有明显品牌符号的物品***如：Nike的对勾，LV的老花，adidas三道杠等具有较强识别度的文字，图案或LOGO | 运动鞋咖啡杯矿泉水瓶 |  |  |
| 重合率*具体评判逻辑查看查重规范 | 使用站外关键词时，需要保证生成图片与站外参考不一致使用垫图时，需要保证生成图片在构图、颜色、元素构思上与所垫图片不一致所有生成（套用关键词/垫图/自行生成）的图片的主要结构需要保证与站外图片不一致//、 |  |  |  |
| **其余禁用元素严格遵循通用规范要求** |  |  |  |  |

## 二.生产要点

### A.生产方式及案例

#### 1.Midjourney（MJ）文生图

文生图中，需要注意prompt词组的合理性，以确保成图能够最大限度贴近需求。以Midjourney举例：

| **素材分类** | **关键词组成规则/逻辑** | **注意事项** | **需求描述** | **撰写示例** | **生成示例** |
| --- | --- | --- | --- | --- | --- |
| 元素 | 主体（必须）+主体状态（如有）+风格（如有）+背景（必须）+比例（必须） | 主体关键词必须准确，生成素材才能准确其他关键词可根据实际情况进行修改 | 一朵可爱感的3D玻璃花 | 【主体】A 3D glass flower（3D玻璃花）【主体状态】translucent（半透明）【风格】cute style, cartoon style, Disney style（可爱风格，卡通风格，迪斯尼风格）【背景】black background（黑色背景）【比例】--ar 1:1 |  |
| 图片 | 主体（必须）+主体状态（如有）+环境（如有）+灯光（如有）+风格（如有）+其他（如有）+比例（必须） | 森林里的木屋 | 【主体】Wooden house in the forest（森林中的木屋）【主体状态】glass facade with lights on（玻璃里面里面开灯）【环境】fog around just after rain, Tyndall effect（雨后雾气弥漫，丁达尔效应）【灯光】natural light（自然光照）【风格】photo（照片）【其他】high quality, 8K【比例】--ar 4:3 |  |  |

#### 2.Midjourney（MJ）图生图

图生图中，需要注意prompt词组对关键词的影响，以及成图与垫图的相似度，如果过于相似，可能会配判定有侵权风险。

| **素材分类** | **关键词组成** | **注意事项** | **需求描述** | **垫图** | **关键词** | **生成示例** |
| --- | --- | --- | --- | --- | --- | --- |
| 元素 | 垫图网址（必须）+主体（必须）+主体状态（如有）+风格（如有）+背景（必须）+比例（必须） | 1.垫图选取尽量不要出现文字，以防后期占比加重2.垫图网址和关键词之间需要加【空格】隔开3.可以通过尾缀关键词 --iw 0.5-2，对垫图进行变化设置（数值越大越接近原图）4.可利用添加seed值，提高生成准确度5.使用垫图时关键词可以尽量简单 | 盾牌3Dicon |  | 【主体】a simple shield,3D icon（一个简单的盾牌，3D图标）【主体状态】blue,frosted glass（蓝色，磨砂玻璃）【风格】technology sense,industrial design（科技感，工业设计）【灯光】natural light（自然光）【背景】white background（白色背景）【其他】3D rendering, Oc renderer, 18k（3D渲染，Oc渲染，18K）【变化】--iw 2 --niji 5 --style expressive |  |
