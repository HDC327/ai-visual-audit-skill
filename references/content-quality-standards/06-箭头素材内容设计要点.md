# 箭头素材内容设计要点
基于，针对箭头素材的设计要点说明

| 箭头类型 | 要点说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 基础几何箭头 | 几何线条规整，线条流畅，做好结构对称和对齐；格式需要做成矢量，方便应用和编辑； |  |  |
| 变体几何箭头 | 基于几何图形延展的箭头组合造型，如正负形结合、多箭头组合、箭头结合其他图形等，造型有设计感，耐看的细节处理；格式需要做成矢量，方便应用和编辑； |  |  |
| 手绘涂鸦箭头 | 随意的手绘涂鸦风格，有明显的笔刷质感；线条流畅，有动感，活泼生动； |  |  |
| 立体质感箭头 | 基于基础箭头的3D化呈现，线条流畅；箭头造型摆放呈现一定角度弯曲或倾斜，增强立体透视；边缘添加高光，使整体质感更精致；渐变色和透明材质应用，丰富材质细节； |  |  |

注：图片来源网络，仅做案例参考，如有侵权，联系删除
