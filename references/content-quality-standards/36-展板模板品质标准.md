# 展板模板品质标准
该标准基于 ，设计时需一并参考

### 一. 设计原则

| **设计原则** | **正面案例** | **负面案例** |  |  |
| --- | --- | --- | --- | --- |
| **和谐统一** | 符合受众调性 | 展板设计需符合受众调性，贴合主题/行业的文化与审美偏好确保整体视觉简洁统一，保持元素风格协调一致，同时避免使用过多字体样式 |  |  |
| 色彩契合主题 | 主色调与行业/主题相符，如科技展板用蓝色增强科技感，年会展板用红金色营造喜庆氛围避免使用过多色彩导致视觉混乱 |  |  |  |
| **层级清晰** | 标题醒目 | 标题醒目，确保在远距离（3-5米）仍清晰可读，可用大字号、加粗、对比色突出（如发布会主标题字号＞副标3倍），副标题、正文、活动时间依次递减 确保标题与背景形成高对比度（如深色背景用浅色字体），避免复杂纹理背景干扰文字识别 |  |  |
| 视觉引导 | 运用引导线条、箭头、图标等元素，帮助受众顺畅地从一个信息块到下一个，确保视觉引导清晰 通过动线设计引导视线流向核心信息 |  |  |  |
| **布局合理** | 留白得当 | 留白得当，避免信息拥挤 |  |  |
| 视觉均衡 | 合理分配文本与图像空间，使整体和谐美观 运用黄金分割、对角线构图等方式，使版面更具美感和稳定性 适当利用倾斜、流动线条等手法，增加画面动感和视觉张力 |  |  |  |
| 版式有序 | 采用网格布局或对称布局，主标题突出，重要信息（如主题、时间、地点）位于视觉黄金区，便于传递信息 |  |  |  |
| 空间感营造 | 构图保持层次感：背景可采用虚实结合方式，如模糊纹理+清晰前景元素，增强空间感 |  |  |  |
| 文字位置优化 | 重要信息位置不可过低，应贴合用户视觉习惯，置于画面中上区域 |  |  |  |
| **传达清晰** | 文字清晰 | 选择笔画清晰识别性强的字体，注意字体不宜过细（展板应用场景多为中远距离且尺寸较大），用颜色对比提升文字的可读性 |  |  |
| **细节丰富** | 质感细节 | 肌理运用：在背景、字体或元素上加入细腻的纹理（如纸感、金属光泽、水墨晕染）提升质感。光影层次：局部光效点缀，如霓虹灯光、折射光线等，使展板更具高级感 |  |  |
| 标题设计 | 除通过字号区分文字层级外，还可结合字重、字间距、颜色深浅等多维度对比，增强设计层次感适当调整部分文字的位置，如轻微旋转、上下错位或交错排列，以增加动感 可尝试部分笔画隐藏、裁切，或让文字与背景元素互动，使标题更具艺术感 （需保证可读性） |  |  |  |
| 元素点缀 | 有辅助性元素点缀，画面丰富不单调 |  |  |  |
| 画质清晰 | 保证高清画质，100%放大后图形边缘**清晰锐化**，无模糊感，无锯齿马赛克，无杂色像素 |  |  |  |
| 易于延展 | 考虑用户体验，提供最大程度的**可编辑性**，如源文件分层细致合理，元素保持完整等，以便素材延展和适配不同场景 | / | / |  |

### 二. 设计要点

不同类型的展板在**色彩、排版、字体、信息呈现**等方面都有不同的设计侧重点

| **展板类型** | **设计要点** | **正面案例** | **负面案例** |
| --- | --- | --- | --- |
| **年会/庆典展板** | **风格定位：**氛围热烈，突出喜庆感和企业文化感**色彩搭配**：以红、金为主，渐变光效或金属质感提升奢华度 **视觉元素**：金箔、光影、丝带、粒子特效、灯光等，增强节庆感与立体空间感 **光影元素**：通过光晕、渐变、折射等提升空间感和氛围感 |  |  |
| **发布会/科技展板** | **风格定位**：现代、科技感强，突出创新、未来感，强调前沿科技氛围 **色彩搭配**：多以蓝、黑等冷色系为主，搭配光效、渐变、金属质感或高亮点缀，营造科技感**视觉元素**：光影渐变、线性流动、HUD界面、数据流、几何元素、科技网格、3D几何形态等，提升未来感与科技感 |  |  |
| **校园活动/社团****宣传展板** | **风格定位**：活泼有趣，贴近年轻群体，信息传播做到轻松易懂 **色彩搭配**：根据活动性质选色，青春活力类可用亮色（如蓝、黄、绿、橙），学术类适用稳重色（如深蓝、灰、藏青），艺术类可尝试撞色、渐变或手绘风。 **视觉元素**：结合活动属性，可使用涂鸦、手绘插画、卡通图形、拼贴风等，增强趣味性与吸引力 |  |  |
| **商务会议/展览展板** | **风格定位**：专业、稳重、现代感强，信息传达精准**色彩搭配**： 常用经典商务色调，辅以低饱和度冷色和金属质感，以传达严谨与高端氛围。避免使用过多鲜艳、饱和度高的颜色**字体选择**：选用现代简洁的无衬线字体或稳重的衬线字体，保持文字的清晰易读，避免使用装饰性字体**视觉元素**：使用简洁的图形元素（如直线、矩形、圆形）、数据可视化图表、线性科技图案等 |  |  |
| **文化宣传/党建展板** | **风格定位**：庄重、简洁、具有文化和历史深度，突出党建理念和文化特色**色彩搭配**：党建类红色为主色调，象征革命精神和热情**视觉元素**：党建类标志性元素、红色文化符号。还可结合中国传统文化符号（如祥云、莲花、书法元素）塑造文化深度和历史感 **情感渲染**：利用色彩、光影效果，突出主题精神（如红色革命、人民力量等） |  |  |
| **公益宣传/社会活动展板** | **风格定位**：主题鲜明，突出公益理念，强调视觉冲击力，增强情感共鸣 **色彩搭配**：采用温暖、明亮的色调，如绿色（环保）、蓝色（健康）、橙色（关怀）、黄色（阳光、希望），搭配中性色（白色、灰色）保持清晰简洁。避免过于沉重的颜色，避免产生压迫感。 **字体选择**：选择温和、简洁的字体，确保文字清晰易读，传递亲和力。 |  |  |
