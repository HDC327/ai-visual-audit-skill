# 文字工具素材品质标准
### 设计原则与标准

| **设计原则** | **标准说明** | **正面案例** | **反面案例** |
| --- | --- | --- | --- |
| **元素完整****​** | 请确保核心视觉元素完整的呈现，避免关键信息被裁切或缺失，保持画面构图的完整性。 |  |  |
| **信息易读** | **文字主导**：请确保文字信息在视觉中占据主导地位，并与其他元素比重上保持协调关系。 |  |  |
| **层级清晰:** 请明确区分标题、副标题、正文等不同层级的文字样式，层级分明，引导阅读视线，预设合理的文字层级关系。​ |  |  |  |
| **风格统一** | **字体与元素****风格一致:** 字体风格应与整体设计风格保持一致，例如：现代简约风格搭配无衬线字体、古典优雅风格搭配衬线字体、手绘风格搭配手写感字体；**功能匹配:** 根据任务功能选择合适的字体；例如：标题类使用醒目的粗体；标签使用易读识别性强的字体；字幕/基础类使用基础款字体。**视觉平衡性:** 字体参数设置强度(大小/字重粗细/圆角大小/色彩搭配等)属性应与元素参数设置强度(大小/线条粗细/圆角大小/色彩搭配等)属性相一致，避免视觉上的冲突或不平衡；确保所有装饰元素的视觉风格统一，形成和谐的视觉整体。**字体与字体****字体数量控制：**同一素材使用的字体数量不宜过多，控制在3种内，避免视觉混乱。**搭配和谐:** 选择风格相近的字体进行搭配，避免使用风格差异过大的字体。 |  |  |
| **配色协调** | **色彩搭配和谐，遵循色彩理论；****明暗:** 利用不同明度的色彩进行搭配，避免沉闷、老旧；**冷暖:** 将冷色调和暖色调进行对比，营造视觉冲击。**纯度:** 将高纯度与低纯度色彩进行对比，突出重点信息。确保整体视觉效果的和谐统一；避免配色偏暗和脏。 |  |  |
| **样式简化** | **整体细节简化****优选语义化图形，强化信息传达：**icon 图标/符号为优先；相较于复杂插画，icon 图标/符号具有结构简洁、语义清晰、识别度高的优势，为文字素材装饰元素的首选。**避免过度装饰：**保持简洁明了的视觉风格，避免喧宾夺主，影响文字信息阅读。**避免尖锐：**边缘不宜过分尖锐，需处理圆滑 |  |  |
| **整体样式明显**任何样式参数的设置都需要明显、突出；举例：描边选择与文字对比明显的描边颜色，以确保文字的可识别性；过粗：文字细节被掩盖，导致识别困难；过细：起不到强化对比的作用且影响整体质量。 |  |  |  |
| **画质清晰** | 100%放大后图形边缘清晰，无模糊感，无锯齿马赛克、无杂色，避免低像素。 |  |  |
| **易于编辑** | 请严格按照执行，对文字、元素、文字样式框(如有)进行正确设置，以确保文字素材编辑性。 |  |  |
