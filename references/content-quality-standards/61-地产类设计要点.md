# 地产类设计要点
**1.固定LOGO位**

logo一般情况下有两个，左边稿定logo，右边打字：稿定置业或稿定地产

需置于画面左右上角，对齐等大；logo大小参见规范内示意大小即可，右边项目logo最小字号为30px​

**2.信息层级区分清晰**

主副标内容在排版上的信息层级，通过字号/字体/粗细来区分

| 整张海报效果 | 主标题最小字号为70px（主标题可用极小英文做排版点缀，英文最小字号为8px） |
| --- | --- |
| 副标题最小字号为36px |  |
| 正文最小字号为30px |  |

**3.产品信息单独展示**

产品信息包含：楼栋号或面积段+产品名

| 规范&案例 | 说明 | PSD附件&预览&说明 |
| --- | --- | --- |
| 产品信息单独一行，与正文之间需有间隔；最小字号为 23px“建筑面积”、“建面约”为不需要被看清，但必须存在的免责文字；最小字号为 8px | 可直接下载右方PSD附件，选择版式进行套用。​<br>前半部分【建筑面积/建面约 xx-xxx㎡】可直接套用，后半部分的产品名，需要根据实际文案变换版式；右方版式仅为参考。 |  |

**4.压条组合内容**

压条组合包含：地址+电话+二维码+免责

| 规范&案例 | 说明 | PSD附件&预览&说明 |
| --- | --- | --- |
| 压条整体排版需规整，整体排版组合为矩形排版地址最小字号22px电话最小字号18px二维码最小尺寸100*100px | 直接下载右方PSD附件，选择版式进行套用，无需更改文案&版式。​<br>注意：是否需要文字背景看画面效果；勿全部有文字背景 |  |
