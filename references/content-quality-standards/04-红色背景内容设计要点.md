# 红色背景内容设计要点
基于 ，针对红色背景的设计要点说明

| 设计要点 | 要点说明 | 图例 |  |
| --- | --- | --- | --- |
| 纹理 | 流沙 | 表现流沙的动态和流动的特质，画面有动态感；丰富细腻的的色彩渐变，画面具有层次感；流沙的动态曲线需符合流体力学，流畅自然，有较为统一的流动趋势；需清晰刻画流沙材质的颗粒细节； |  |
| 笔刷 | 有清晰的笔刷纹理，能分辨得出笔刷走势和造型；有流畅的线条和优美的造型；有明显的笔触效果或水墨水彩的晕染效果；笔刷颜色与整体背景融合； |  |  |
| 粒子 | 由粒子均匀或有规律地分布，组成规律或抽象的造型；粒子以噪点的形式均匀分布平铺于画面中，颜色与整体背景融合；粒子需清晰且颗粒细密，有层次感； |  |  |
| 丝绸 | 表现丝绸柔软光滑的质感；需要有优美的动态曲线和自然的褶皱起伏，褶皱不宜过多，丝绸整体应是平滑的；刻画高光阴影，需有明显丝绸布料纹路； |  |  |
| 抽象几何 | 线条和形状分布具有几何规律美感，如对称、等距、等差排列、S曲线、抛物线、黄金分割等；使用线条或色块构成，整体画面视觉平衡有序，有空间感和层次感； |  |  |
| 光影 | 光线 | 使用高光线条引导视觉焦点，创造画面深度和空间感；光线流畅，多条光线同时出现时需要有明显主次关系；光线边缘色彩过渡平滑自然，强调发光的效果；光线的动态效果，增加光线的闪烁、移动或变化来让画面更具动感；通过光的散射、反射、光晕光斑等效果，增强光线的真实效果； |  |
| 投影 | 模拟自然光通过窗户或者缝隙投射在平面上的光影效果；投影形状造型简洁，边缘过渡柔和，离光源越远的形状边缘越模糊；投影角度统一，通常为倾斜角度，与画面构图平衡； |  |  |
| 场景 | 展台场景 | 画面有明显的展台台面，台面平台空间足够大，方便放置产品；台面上方高度应展画面2/3以上；多个展台需有高低错落；展台角度为平视稍微往上，能看到台面空间；透视需准确，需体现立体感和空间感；装饰元素不宜过多，适当点缀装饰在台面周围或背景上； |  |
| 主题场景 | 有具象元素构成的主题场景，元素作为底纹或点缀装饰；表现场景宏大，多用广角大画幅的画面构图；使用金色、白色元素，与红色背景搭配和谐； |  |  |
