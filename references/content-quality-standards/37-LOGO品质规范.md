# LOGO品质规范
#### 基础视觉标准

| **设计原则** | **设计要点** | **说明** | **图例** |
| --- | --- | --- | --- |
| 协调统一 | 调性应与品牌定位、价值观和目标受众保持一致受众认知、色彩组合、字体风格统一 | 一致性有助于建立品牌信任和认同感。如果Logo与品牌形象不匹配，可能会让消费者感到困惑，削弱品牌影响力。 |  |
|  | 整体比例协调、平衡 | 注重图形的比例、对称、平衡、节奏等形式美法则，使图形具有美感和稳定性，给人以舒适的视觉感受。平衡可以通过大小、形状、颜色等元素的分配和组合来实现。一个平衡的logo能够传达品牌的信心、稳定和专业性。比例可以运用网格系统、黄金比例法来保证各个元素之间的大小关系比例协调。一个正确的比例可以帮助构建一个和谐的视觉效果 |  |
| 层次分明 | 符合LOGO构成要素的组合顺序图形符号标准字标语 | 图形符号（字标/图标）是第一视觉焦点，适合快速建立品牌联想。标准字（品牌名）作为第二优先级，确保可识别性（部分极简设计可能将标准字与图形符号合并为一体，如苹果、耐克）。标语（品牌口号）通常作为辅助元素，以较小字号或轻量化设计出现，用于强化品牌理念或功能定位。 | ​ |
| 传达清晰 | 准确传达品牌信息核心业务、特点、价值观等 | 通过提炼核心元素，使得消费者能够对品牌有一个初步的认知和联想，建立起品牌与消费者之间的情感联系。 |  |
|  | 确保能够快速识别 | 标志的辨识度要保证在不同环境的一定距离内，标志依然能够轻易辨认，避免出现字体粗细、负空间间隙、色彩对比、清晰度等因素带来的设计问题。 |  |
| 简洁明了 | 提炼复杂繁琐的图形除特殊化类型（如徽章） | 简洁的LOGO更容易被识别、记忆和传播，去除不必要的复杂元素，以简洁明了的图形、文字或组合来传达品牌的核心信息，使人们在短时间内能够快速理解和接受。*百事LOGO演变案例 |  |
| ​ | 控制色彩的数量 | 使用2-3种主色，避免过多色彩导致视觉混乱。 |  |
| 长远持久 | 设计不过时的LOGO | LOGO设计不盲目追随时下流行设计手法，而以更长远的前瞻性视野去看待，避免因短期流行趋势而频繁更改。 |  |

#### 优质视觉标准

在基础视觉标准之上，增加要求。

| **设计原则** | **设计要点** | **说明** | **图例** |
| --- | --- | --- | --- |
| **细节丰富** | 超精细、严谨的制图 | 高质量LOGO的细节丰富性体现在系统性的精致化，通过数学精度、工艺适配、层级控制等多维度细节设计，使符号在不同场景下都能呈现专业质感。这种细节不仅提升视觉品质，更通过可探索的隐藏元素增强品牌记忆点。 | *高精度LOGO案例 |
| **创新创意** | 追求独特的外观和个性 | 高质量的LOGO在符号创新上，创造行业标志性符号，能通过符号变形传递品牌核心理念（如工商银行将镂空“工”字与圆形方孔钱币结合）；在语义深度上，常通过植入隐藏式负形、多维度隐喻与双关设计来传递品牌哲学。（如Adidas三叶草同时象征运动轨迹与地球） | *工商银行案例 |

#### 易用性标准

| **设计原则** | **设计要点** | **说明** | **图例** |
| --- | --- | --- | --- |
| **易于延展** | 适应不同媒介和环境 | Logo设计应考虑不同媒介和应用场景的适应性，包括不同尺寸、色彩模式和背景环境。如网站、社交媒体、宣传册、产品包装、建筑物标识等 | *微软必应案例：以手机应用图标进行延展性测试，左侧老版logo有4种延展的可能性，用b作为品牌元素但没有任何特点，用后两个在手机端很小，严重影响辨识度。新版logo相对来说，标志至少解决了延展性的问题。 |
