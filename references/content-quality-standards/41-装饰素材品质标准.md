# 装饰素材品质标准
### 设计原则及标准

| 设计原则 | 标准说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 元素完整 | 确保主体元素完整，没有缺失的部分。构图合理裁剪，不影响素材应用。 |  |  |
| 线条流畅 | 线条应该流畅自然，避免出现生硬或断裂的线条。 |  |  |
| 色彩协调 | 色彩搭配和谐，遵循色彩理论，以确保视觉效果的和谐统一。 |  |  |
| 材质细腻 | 准确细腻的的材质表现，适当增加纹理丰富视觉层次感。 |  |  |
| 造型简约 | 避免复杂的造型结构，主体元素需有较好的识别性。 |  |  |
| 美观实用 | 视觉美观，不需做过多设计，简单直接，以实用为主。单色或渐变填充的图形，尽可能使用矢量图形格式，保留素材可编辑性。 |  |  |
| 画质清晰 | 保证高清画质，100%放大后图形边缘清晰锐化，无模糊感，无锯齿马赛克，无杂色像素。 |  |  |

注：图片来源网络，仅做案例参考，如有侵权，联系删除
