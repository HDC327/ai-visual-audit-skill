# 中秋素材内容设计要点
## 艺术字

在 基础上做的一些补充

| **类型** | **要点说明** | **正面图例** | **反面图例** |
| --- | --- | --- | --- |
| 毛笔艺术字 | 整体有节奏感，字体大小有变化，笔画粗细有变化；毛笔笔触明显，特别是拖尾；构图合理，整体紧凑不松散，视觉上协调稳定；装饰元素品质高，无拼凑感； |  |  |
| 电商艺术字 | 需做对应字形设计，而非直接使用字库字体；文字材质明显，非扁平化处理；构图合理，整体紧凑不松散，视觉上协调稳定；装饰元素品质高，无拼凑感； |  |  |

## 元素

| **类型** | **要点说明** | **正面图例** | **反面图例** |
| --- | --- | --- | --- |
| 兔子 | 兔子形象可爱；比例协调；动作灵动不呆板僵硬；光线柔和，不要太暗太脏；毛绒的质感要细腻真实； |  |  |
| 月亮 | 光感明显；纹理不宜太深太重；构图易用，前景树枝不宜过短； |  |  |
| 灯笼 | 光感明显；造型优美，有飘动态感；构图错落有致，不呆板；配色通透不暗沉； |  |  |
| 嫦娥 | 比例协调；服饰飘逸；动作优美；画风配色新颖不陈旧； |  |  |

## 背景

| **类型** | **要点说明** | **正面图例** | **反面图例** |
| --- | --- | --- | --- |
| 弥散渐变 | 构图留白，有层次感，有视觉焦点；颜色渐变层次丰富，用渐变做出体积感、光感；整体色调协调统一，不违和不拼凑； |  |  |
| 中国风 | 构图留白，有层次感，有视觉焦点；色调及氛围，呈现中式意境；画风准确，不混搭； |  |  |
| 合成/3D | 突出节日氛围感，温馨圆满；画面有足够光感，层次感；元素精度高质感强； |  |  |
