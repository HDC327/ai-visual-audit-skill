# 双十一素材内容设计要点
**艺术字**

在 基础上做的一些补充

| **类型** | **要点说明** | **正面图例** | **反面图例** |
| --- | --- | --- | --- |
| 书法字 | **笔触干净，字形流畅：**：笔画明确走势顺畅，结构自然舒展**节奏有变化**：整组字之间有张有弛，富有节奏感；**构图紧凑稳重**：保持统一重心，避免松散漂浮 |  |  |
| 3D字 | **构图完整，有整体感**：字体位置应居中稳固或形成主视重心，搭配的元素、底座、光效要共同构成一个稳定画面，而不是几个独立物件堆在一起； **材质表达精细，突出质感**​**字与元素统一和谐；**红包金币类装饰元素需避免喧宾夺主、遮挡主字或出现割裂拼贴感 |  |  |
| 数字 | **构图有层次，主次分明**：避免数字全部同样大小、等距排列，可通过字号差异、倾斜角度、间距变化等营造节奏感**视觉重心清晰**：控制字体布局，防止画面头重脚轻或数字密集堆叠 |  |  |

**元素**

| **类型** | **要点说明** | **正面图例** | **反面图例** |
| --- | --- | --- | --- |
| 通用 | **色彩鲜亮**：推荐多巴胺系明快配色营造热烈的促销氛围 **造型合理**：元素本身形态必须完整，手脚造型比例合理**材质有质感，**透明材质必须通透（如玻璃、亚克力），避免塑料感或质感模糊；表面需有真实光泽与细节变化（如边缘高光、局部阴影）**可爱有亲和力**：拟人类元素加入眼睛、表情时，需自然有情绪，避免翻白眼 |  |  |
| 组合类 | 具备氛围和情节感，避免元素孤立堆砌构图要有冲击力和空间感：通过动线、位置、节奏建立视觉焦点，通过光效、粒子、投影等辅助表现空间感 |  |  |
| 红包类 | 厚度比例合理，结构完整清晰飞出路径顺畅，有节奏（可使用弧形、螺旋、放射状等自然路线，红包间距有疏有密）透视统一，不扭曲 |  |  |
| 礼盒类 | 盒体、盒盖、缎带等结构清晰，层级分明造型圆润饱满，避免边角太锐或方正僵硬 |  |  |
| 金币 / 优惠券类 | 造型标准，边缘干净，有一定厚度漂浮状态元素动态感强，虚实过渡自然 具备空间层次感，大小对比清晰布局有节奏，避免平均撒满 |  |  |
| 标签 / 丝带类 | 轮廓自然，有体积感，倒角处理圆滑具备适当卷曲/翻折变化 |  |  |

**背景**

| **类型** | **要点说明** | **正面图例** | **反面图例** |
| --- | --- | --- | --- |
| 氛围背景 | **色彩有层次，画面有呼吸感**：避免纯色铺满，注意留白与节奏感 **构图有空间，元素不过多** **氛围适度强化**：可加粒子、光效等，但方向统一、不过量 |  |  |
| 展台背景 | **视角统一、构造清晰**：建议使用 45° 或正视角，展台结构完整，有体积感**材质干净、氛围明亮**：避免脏灰色调；画面色彩明快**构图合理、留白适度**：展台大小适中，商品摆放空间明确，不贴边不挤角**光影自然、背景干净**：光源方向明确，投影真实；背景简洁，避免杂乱 |  |  |
