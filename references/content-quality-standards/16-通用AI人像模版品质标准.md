# 通用AI人像模版品质标准
#### 一、人像品质标准

| **品质要点** | **要点说明** | **正面案例** | **负面案例** |
| --- | --- | --- | --- |
| **契合主题** | 人物特征设计需与主题行业相符，不同的行业使用不同的风格的人像，美业相对露肤度较高，讲师/喜报多商务风格，应援/服饰箱包时尚等 |  |  |
| **真实自然** | AI生成人像需尽量体现皮肤毛孔纹理等细节，通过自然表情、舒展动作、真实场景增强观者共鸣，避免过度美化导致失真。 |  |  |
| **视觉聚焦** | 以人像为核心，通过画质清晰度、色彩对比、构图设计引导视觉重心。​ |  |  |
| **画质清晰** | 最好生成1.5-2倍以上的图，抠图后转智能对象再进行调整，以保证素材的高清（可以先1倍图确认可用素材后再2倍图生图） |  |  |
| **色彩搭配** | 注意色彩的对比和协调性，契合行业及主题调性，避免灰、暗、脏、过曝等色彩不协调问题 |  |  |
| **抠图精细** | 人物边缘，如发丝/毛绒等扣图细致不生硬可借助 [稿定AI抠图(人像模式)](https://www.gaoding.com/koutu)或 [豆包](https://www.doubao.com/chat/create-image)（免费）扣图 |  |  |
