# 头像模板品质标准
### 基础视觉标准

| **设计原则** | **要点** | **说明** | **正面案例** | **反面案例** |
| --- | --- | --- | --- | --- |
| **和谐统一** | 文字 | **选择与主题和风格相匹配的字体，来传达情感或强化信息**商务类➡ 选择与整体风格匹配的**庄重正式**的字体，通常使用衬线字体如宋体、无衬线字体如黑体，确保文字清晰可读，避免复杂或花哨的字体应援类➡ 根据不同的视觉氛围选择匹配的字体，如华丽感的画面可选择衬线字体；现代感的选择字重较粗的无衬线字体；综艺感的选择粗大趣味感字体等搞怪类➡ 选择卡通、综艺类字体，可使画面更活泼生动有趣外卖类➡ 根据不同的餐饮类型选择字体，如中式餐饮选择粗大的毛笔字体，现代高端餐饮可选择无衬线字体 |  |  |
| 图像 | **选择合适的图像，能够直观地传达海报主题****，****图像选择需要契合用途类型**如商务需穿着正装、应援需穿着时尚，外卖选择具有吸引力的产品作为展示图**情侣头像**需具备**关联性**，如同样的元素、同样的动作、同样的画风等，或者连续的拼图 | **情侣头像** |  |  |
| 色彩 | 色彩不仅影响**视觉吸引力**，还能**传递情感和信息**商务类➡ 多为奢华的白金、蓝金、黑金、红金等去凸显专业的商务感应援类➡ 色彩丰富，可根据人物着装配色去延展外卖类➡ 餐饮类色彩多为暖色调，尤其是中式餐饮、正餐类；健身餐可选择绿色系等 | 商务类应援类外卖类 |  |  |
| 背景 | 商务类➡ 女性可选择亲和的浅色背景，男性可选择庄严的深色背景，避免复杂或杂乱的背景分散注意力应援类➡ 符合应援调性，如光感背景、格纹背景等插画类➡ 背景干净，留白得当，突出人物 | 商务类应援类插画类 |  |  |
| **层次分明** | 文字 | 通过**字号、字重、字色、字效****、添加图形**等来区分标题、副标题和正文等信息的**层级结构** |  |  |
| 排版 | 使用**版式亲密性原则**来梳理层级关系，并用**对比、层级**来区分整体画面的层次商务类➡ 左右结构排版或半包围式版式应援类➡ 分为前中后3个层级，一级文案、二级人物、三级背景搞怪类➡ 半包围、全包围、上下结构等，版式比较自由 | 商务类应援类搞怪类 |  |  |
| **焦点强调** | 文字 | 通过**颜色、材质、字号、字重、字效、字体混搭、添加图形**等方式来**强调核心文案**信息 |  |  |
| 图形 | 可通过**醒目的图形、大胆的图像**作为视觉焦点，作为吸引注意力的核心区域 |  |  |  |
| **传达清晰********** | 图像 | **清晰的抠图、****适当的裁剪比例**商务类➡ 腰部以上凸显正装正式感应援类➡ 突出人物脸部可取胸部以上外卖类➡ 具有吸引力的产品作为展示图 |  |  |
| **简洁明了****** | 文字 | 文案使用**简短的句子**和**强有力的词汇****直接传达核心信息，**避免过多文字堆砌，保持简洁明了；字体选择一般**不超过3个字体**，以免分散观众的注意力商务类➡ 直接说明你是谁以及你的title/提供的服务/产品是什么。比如：“如专注于为女性提供天然护肤品、抗衰美肤专家”等。应援类➡ 使用简短但有力的文字表达支持，应援类如“xxx加油！”、“出道x周年”等，生日类如“xxx生日快乐”等。外卖类➡ 宣传语可以突出产品的特色（如“用鲜鸡才好吃”）、提供的优惠（如“第二份半价”）或品牌的实力（如“门店数量突破xxx家”）。 |  |  |
| 图像 | 头像通常的展示面积是很小的**，需避免使用过度复杂/杂乱的图像**，以免分散注意力 | **​** |  |  |
| **留白得当** | 排版 | 需考虑需求调性，利用**版面率知识，创造出相匹配的画面气质**商务类➡ 低版面率-留白多，画面气质更素雅、庄严、安静应援类/外卖类➡ 高版面率-留白少，画面气质更活泼、热闹 |  |  |

### 优质视觉标准

在基础视觉标准之上，增加要求。

| **设计原则** | **要点** | **说明** | **案例** |
| --- | --- | --- | --- |
| **细节丰富** | 文字 | 使用填充（颜色/图案）、描边、错落、倾斜、变形、手写字、光效、添加装饰等方式来**丰富字体效果** |  |
| 图形/图像 | 可爱感的头像可以使用丰富饱满的卡通元素增加视觉氛围，丰富画面细节 |  |  |
| 排版 | 通过给文本**添加点线面、色块、图形、符号等**元素来丰富细节，巧用**相似或反差字体进行混搭**，使版式得到优化 |  |  |
| **创新创意** | 文字 | 根据作品主题、受众对象的特征，通过**创意字体、创意文案**，来吸引受众对象的关注 |  |
| 图形/图像 | 通过**同构、异构、解构、重构、拟人、矛盾、悖理、变异、特殊材质、新颖特殊的画风等创意手法**，创造出创意的画面 |  |  |

### 易用性标准

| **设计原则** | **要点** | **说明** | **案例** |
| --- | --- | --- | --- |
| **易于延展** | 图形/图像 | **头像描边、投影**需使用编辑器进行还原，保证编辑可用性**图片描边**：根据视觉稿效果选择对应描边样式说明:除个性化风格外，需尽量设置还原 |  |
| 色彩 | 但凡涉及到矢量图形(基础线 条/色块/图形/图框/icon等)都需保留矢量可编辑，便于用户修改颜色。**编辑器内需检查是否能修改颜色** |  |  |
| 排版 | **合理分组**，同层级文案或临近元素打组，主要目的是方便用户进行元素间的整体移动和调整提高使用体验 |  |  |
