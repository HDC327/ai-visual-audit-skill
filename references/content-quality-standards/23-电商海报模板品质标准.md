# 电商海报模板品质标准
该标准基于 ，设计时需一并参考

| 设计原则 | 设计要点 | 说明 | 正面案例 | 负面案例 |
| --- | --- | --- | --- | --- |
| 和谐统一 | 品牌风格统一 | 确保海报风格与店铺整体风格一致，避免因海报设计与店铺风格冲突而降低用户信任感。 |  |  |
| 目标统一 | 无论是横版还是竖版海报，设计都要紧密围绕营销目标，如推广新品、促进促销、提升品牌形象等。所有设计元素都应服务于这一目标，确保海报能够有效传达信息并引导消费者行为。 |  |  |  |
| 色彩搭配统一 | 选择符合品牌调性和产品定位的色彩方案，避免过于刺眼或混乱的色彩组合。色彩应具有视觉冲击力，同时保持整体协调性。 |  |  |  |
| 层次分明 | 明确信息层级 | 明确信息的主次关系。在海报中，最重要的信息（如产品名称、活动主题）应该放在最显眼的位置，字体大小和颜色可以与其他信息区分开来。次要信息（如产品特点、活动规则）可以放在相对次要的位置，但也要清晰可见。 |  |  |
| 焦点强调 | 突出核心卖点 | 明确产品的独特卖点（USP），如功能、价格优势、品质、品牌故事等，并在海报中突出展示。促销海报需突出折扣、满减、赠品等关键信息。 |  |  |
| 传达清晰 | 文字简洁易读 | 确保海报上的文字清晰可读，选择合适的字体、字号和颜色对比度。文字内容要简洁明了，避免使用过于生僻或难以理解的词汇。 |  |  |
| 促销信息醒目 | 如果海报包含限时促销或活动规则，需确保这些信息清晰可读，通过大字号、醒目颜色展示，同时用小字号补充说明活动时间、参与条件等细节。 |  |  |  |
| 引导用户行为 | 明确海报的引导目标（如加入会员、参与活动、领券等），并通过明显的按钮设计（如“进店选购”“立即购买”“查看详情”）和清晰的箭头引导来促进用户操作。 |  |  |  |
| 简洁明了 | 文字简洁明了 | 避免过多的文字堆砌，只保留关键信息（如产品名称、卖点、促销信息等）。文字内容需简洁、易懂，避免用户产生阅读疲劳。 |  |  |
| 色彩简洁明快 | 可以使用对比色或渐变色来增加视觉层次感和吸引力。但色彩数量不宜过多，以免造成视觉混乱。 |  |  |  |
| 留白得当 | 元素布局协调 | 避免画面过于拥挤，合理安排元素，留出足够的空白。构图需符合视觉流程原则（如黄金分割、对称或平衡），引导用户的视线自然流动。 |  |  |
| 构图布局要点 | **横版海报**：适合展示宽广的场景、多个产品并列或强调水平方向的信息流动。可以采用左右构图、上下构图或中心构图等方式，合理安排产品、文案和背景元素的位置，保持视觉平衡。 |  |  |  |
| **竖版海报**：更适合突出单个产品、垂直方向的信息展示或引导用户从上到下浏览。常见的构图方式有居中垂直排列、对角线构图等，要注意避免元素过于拥挤，留出适当的空白区域。 |  |  |  |  |
| 创新创意 | 文字类创意 | 主题类文字可用**新颖的材质/风格**表现 |  | / |
| 图像类创意 | 用户在电商平台的浏览时间较短，海报需在短时间内抓住用户眼球。通过鲜明的色彩对比、大胆的构图或独特的视觉元素吸引用户停留。 |  | / |  |
| 使用超现实场景增加海报的独特性和吸引力，但要背景要简洁一点，避免过于复杂，注意背景跟产品可拆分 |  | / |  |  |
