# 马年素材内容设计要点
### 艺术字

在 基础上做的一些补充

| **类型** | **要点说明** | **正面图例** | **反面图例** |
| --- | --- | --- | --- |
| 马形象艺术字 | 文字与马形象结合，需识别出来是马；马的姿势要帅气，笔画有力；整体字形要有识别性，保留文字笔画特征； |  |  |
| 毛笔艺术字 | 整体有节奏感，字体大小有变化，笔画粗细有变化；毛笔笔触明显，特别注意拖尾效果；构图合理，整体紧凑不松散，视觉上协调稳定； |  |  |

### 元素

| **类型** | **要点说明** | **正面图例** | **反面图例** |
| --- | --- | --- | --- |
| 马IP | 马特征明显，不要像驴或狗，注意嘴部和鬃毛特征表现；不要做成西方的独角兽；要营造出春节喜庆氛围；形象要可爱：头身比例Q版，头大身体小；头部圆润，脸颊有一定弧度表现；五官比例可爱，不要过于拥挤；眼睛圆润有神，注意眼神光刻画；不要画成斗鸡眼；鼻子和鼻孔弱化；表情动作要生动不僵硬； |  |  |
| 马插画马纹理 | 马特征明显，重要的特征如马头、马鬃毛、马尾巴、马蹄要准确，一眼看出是马的形象；线条流畅，比例协调，可适当夸张化，但整体要美观和谐；动作灵动，体现马的奔跑或抬腿姿态，不要呆呆地站着；可通过鬃毛尾巴的飞舞状态表现动感；构图适当加入角度透视增强立体感；整体动作要帅，有气质，让人感觉有神性；不要做成西方的独角兽； |  |  |
