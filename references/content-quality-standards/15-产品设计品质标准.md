# 产品设计品质标准
#### 产品品质标准：

⚠️注意：产品不要直接使用白膜

| **设计原则** | **设计要点** | **正面案例** | **反面案例** |
| --- | --- | --- | --- |
| **产品清晰度** | 需保证产品轮廓锐利、logo、纹理等关键细节放大可辨，无噪点与模糊感； |  |  |
| 反光适配材质：金属强反光、磨砂弱反光、织物无明显反光，还原局部反光增强真实感。 |  |  |  |
| **产品立体感** | 通过视觉层次差异，让扁平图片呈现 3D 空间感，避免 “纸片化”。明确产品明暗分区、强化体积感； |  |  |
| **产品质感** | 质感是用户对产品材质的 “视觉预判”，按材质优化质感（软质蓬松、硬质坚硬、液膏流动），避塑料感，实现视觉触达；拒绝塑料感； |  |  |
| **产品色** | 通过丰富产品本身的配色层次，进一步提升画面的精致度与视觉表现力； |  |  |
| **产品信息****​** | 通过细化产品包装呈现 —— 补充 LOGO、产品名称、元素图标等包装细节，强化视觉信息的完整性与层次感，从而提升画面的精致度与专业质感。设计时需符合下述要点：​ |  |  |
| **产品信息没有可编辑限制**，可使用AI工具，根据上述产品信息设计要点直接生成。当AI生成信息出现文字结构错乱，导致无法阅读时，需手动修改，调整至可读状态。**（修改方式：可用AI直接改字或使用AI消除后，再用稿定免费字体补充信息）** |  |  |  |

#### 行业产品特征：

| **行业** | **产品特征** | **示例** |
| --- | --- | --- |
| **五金** | **金属材质表现**（如不锈钢、铜、铁）：需强化其专属质感特征 —— 通过光泽层次、拉丝 / 磨砂纹理的细节刻画，传递坚硬、耐用的视觉感知；**表面工艺呈现：**重点体现防锈涂层的均匀性，同时还原电镀、喷涂等工艺带来的金属光泽层次；**结构细节精度：**突出零件的规整性（如螺丝螺纹的清晰纹理），确保核心功能结构的视觉清晰度，避免模糊或变形。 |  |
| **保健** | **按功能聚焦：**包装一般会通过视觉符号传递功能（如护肝用 “绿叶 / 肝脏图形”，关节养护用 “关节图标”）。**材质质感：**多采用瓶身（玻璃 / 塑料）+ 纸盒的组合，纸盒常用哑光 / 覆膜工艺提升高端感。**通用信息：**品牌标识（Logo）、产品名称（需突出视觉优先级）、辅助装饰元素（需适配品牌调性且不干扰核心信息）、产品规格（清晰标注容量 / 数量等关键参数）。” |  |
| **美妆** | **材质细节的精准刻画****磨砂玻璃**：需呈现半透明雾面质感，瓶身以侧光营造柔和朦胧感，严格规避光面玻璃的刺眼反光，避免廉价感；**金属材质**：重点还原拉丝纹理的细腻层次，光线下需呈现自然的渐变光泽（如鎏金从浅到深的过渡），杜绝塑料仿金属的生硬反光；**塑料材质**：明确区分两类质感 —— 哑光塑料需做到表面柔和反光、触感柔和；覆膜光面塑料则保留轻微、不刺眼的通透反光，平衡成本与质感。**信息层级的视觉呈现：**核心信息需按优先级清晰排布：品牌标识（Logo）、产品名称、规格参数占据视觉核心区域；功效图标等辅助元素作为补充，可搭配简约英文装饰（如 GD、GAODING 等）提升调性。**字体的适配选择：**精致衬线字体（适配高端 / 优雅风格）无衬线字体（适配科技 / 简约风格）避免厚重字体破坏美妆包装的轻盈质感。 |  |
| **家居洗护** | **材质质感：平衡实用与质感****玻璃材质：**带细腻竖纹、半透明雾面（侧光无刺眼反光），金属泵头还原拉丝纹理与自然光泽；**哑光材质：**柔和反光、触感细腻，光面塑料轻微通透反光不刺眼，杜绝塑料 “油腻感”；**软管材质：**带轻微自然肌理，避免生硬褶皱，还原挤压后自然弧度。**核心信息层级：**正面黄金位置突出品牌 Logo、加粗产品名称及核心功效（配简约图标）；侧面以小字号浅色调标注规格、成分表、使用说明，排版整齐不干扰核心信息。**装饰与字体：**优先无衬线字体，高端线可选纤细衬线体；避免复杂艺术字与冗余装饰，保持画面简洁规整，契合洗护洁净属性。**特殊性：**家居洗护包装需绑定核心卖点，如主打 “花香” 可融入花瓣纹理、花卉线稿等对应元素，强化视觉传递。 |  |
| **数码家电** | **功能优先：**明确按键、接口、屏幕等功能区域，包装 / 产品本体标注核心参数（如 “5000mAh”“4K 分辨率”）；**材质质感：精准还原 “科技感 + 耐用性**”**金属材质**：拉丝纹理细腻均匀，冷调金属光泽，边缘无毛刺，喷砂区域哑光不沾指纹；避坑：杜绝塑料质感反光、纹理模糊断裂；**塑料材质：**哑光款带细微颗粒肌理、反光柔和；高光款轻微通透反光、无划痕；避坑：避免廉价颗粒感，明确功能区域材质差异；**字体与符号：**优先无衬线字体，清晰易读；功能图标简洁具象（如 Wi-Fi、蓝牙图标），颜色与机身协调。 |  |
| **零食饮料** | **细分类：**常用非衬线字体，清爽干净，除特殊白酒类别**宠物食品**：主打 “天然 / 高肉含量”，色调清新自然；生图需还原冻干 / 肉粒的真实肉色与纹理质感。**饮料**：玻璃瓶要透明通透，精准还原液体原色；铝箔包装需体现光泽感，可搭配原料图形（如鲜果）强化新鲜度。**生鲜**：突出 “新鲜饱满”，色调鲜艳写实，重点展示食材本身的表皮纹理、果肉质感。**零食**：多用罐 / 包装袋包装，包装需清晰呈现原料信息，生图要体现内容物的丰富堆叠感。**酒类**：色调沉稳厚重，瓶身强化质感 —— 玻璃款突出通透光泽，陶瓷款做磨砂 / 鎏金效果；瓶盖还原金属拉丝质感，字体优先选书法、衬线等古典风格。 |  |
| **服饰箱包** | 禁止出现品牌 Logo；服饰无模特展示时，需通过动态呈现（如自然垂坠、轻微褶皱）凸显面料质感；重点特写面料纹理、五金工艺、缝线细节，避免不同面料质感混淆。 |  |
