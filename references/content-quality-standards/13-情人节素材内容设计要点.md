# 情人节素材内容设计要点
## 艺术字

基于 ，在字体风格、肌理质感、配色及装饰元素上体现唯美浪漫氛围感。

| 设计要点 | 要点说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 字体风格 | 字体以唯美浪漫感和可爱感的字体风格为主，如细线手写体、书法体、曲线花体、卡通体；注重文字识别度和笔画线条流畅；手写字注意文字大小错落，添加连笔与拖尾； |  |  |
| 肌理质感 | 使用立体笔刷增强文字肌理质感和立体感；在笔画末端加入模糊渐变、粒子飘散等效果，增加字体的动态感；3D字体需体现对应材质特点，如金属材质有明显光泽感、丝绸材质的纹路和柔软缥缈感； |  |  |
| 配色氛围 | 以粉色系配色为主，如粉红、粉紫、粉蓝，渐变搭配白色或亮黄色；字体造型结合花瓣、爱心等装饰元素，营造唯美浪漫的氛围； |  |  |

## 背景

基于 ，利用爱心形状、花朵等元素，组成弥散渐变背景或虚拟场景，用色彩和光影营造浪费氛围感。

| 设计要点 | 要点说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 弥散渐变 | 配色以粉红色为主色调，辅以浅蓝、浅紫、浅黄和白色，画面明亮清淡；注意色彩之间相互影响和晕染，过渡柔和；晕染程度需要有变化，可随着造型走势做晕染半径的顺滑变化；适当添加噪点，增强画面质感和色彩丰富度；使用磨砂玻璃或长虹玻璃质感，增加画面层次感； |  |  |
| 虚拟场景 | 注重画面氛围表现，画面主体聚焦，色彩协调，光影统一，构图平衡；以爱心形状为主体延展的场景，心形饱满圆润，以立体风格为主，材质明显；以花朵（玫瑰）为主体延展的场景，花朵盛开，形态优美，颜色以粉色红色为主，注重光感表现；粉色系元素组成的街道或展台场景，用真实的质感和造型表现梦幻的场景； |  |  |

## 元素

| 设计要点 | 要点说明 | 正面图例 | 反面图例 |
| --- | --- | --- | --- |
| 爱心 | 爱心造型需要饱满圆润，线条顺畅，质感清晰，色彩柔和；极简图案类的爱心要求线条流畅，作为装饰元素，造型足够简洁；除非特殊要求，否则一律以弧线为造型线条； |  |  |
| 花朵 | 以玫瑰花为主，花朵造型需要优美，花瓣结构清晰，符合自然花朵绽放形态；质感明显，玻璃质感或透明渐变质感或写实质感，都需表现对应材质特点；配色粉嫩鲜艳，表现花朵新鲜娇艳的状态； |  |  |
