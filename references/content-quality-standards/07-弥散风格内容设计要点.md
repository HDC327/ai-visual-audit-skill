# 弥散风格内容设计要点
### 什么是弥散风

**弥散风格‌定义：**

弥散，直译为“弥漫散开”，意味着向四周扩散、分散。弥散海图案通常从实到虚，单边或周边散开，由单色或多色模糊渐变混合，形成一种虚实结合的有色光感效果。这种风格具有模糊、噪点、朦胧、虚化、暧昧等特点。

**弥散风格‌特点：**

**色彩渐变**：多色渐变自然融合，产生柔和过渡或酷炫的流动效果

**虚实结合**：模糊边缘保留部分清晰轮廓，增强动态感和层次感

**噪点质感**：通过添加颗粒感提升画面复古气质与细节丰富度

​**视觉聚焦**：通过对比色或模糊背景突出核心信息，引导用户注意力

### 设计要点

#### 色彩渐变

| **邻近色**是指rgb色环中，邻近的两种颜色例：蓝-紫，红-黄橙，绿-青，橙-黄等​<br>**互补色**指的是rgb色环中，180 度相对的两种颜色例：蓝-黄，红-青，红-绿 |  |
| --- | --- |

| **设计要点** | **案例** |
| --- | --- |
| **邻近色/相近色渐变**rgb色环中，邻近的两种颜色渐变 |  |
| **互补色渐变**1.互补色颜色搭配大都为冷暖搭配例：红-绿，红-青，黄-蓝，紫-黄等​<br>2.互补色颜色对比明显，直接过渡渐变中间过渡颜色会比较偏灰，有时也可加入过渡色来使画面更为干净例：黄和蓝色直接过渡渐变例：蓝黄中间加入了青色作为过渡（在rgb色环中青色为黄蓝中间色，互补色的中间颜色可以起到过渡作用）​ |  |
| **多色渐变**色彩≥3种，冷暖搭配，相互交融，色彩之间过渡和谐。 |  |
| **暗色背景颜色渐变**暗色背景下，使用颜色一般为高饱和度、明亮配色 |  |

#### 模糊设计

| **设计要点** | **案例** |
| --- | --- |
| **径向模糊**中间对焦区域清晰，边缘模糊，虚实过渡平滑自然 |  |
| **线性模糊**一侧相对清晰，一侧弥散模糊，虚实过渡平滑 |  |
| **毛玻璃遮挡虚化表达**借助毛玻璃/长虹玻璃特性，模拟虚化视觉 |  |
| **虚实结合**保留元素的基本特征轮廓清晰，其余部分概括虚化处理。例：礼盒丝带部分，保证轮廓清晰的情况下有实有虚，过渡自然；盒身部分大胆虚化，简单概括。 |  |
| **模拟镜头虚化**中间对焦的主体清晰，其余元素模糊 |  |
| **模拟微缩摄影虚实表达**例:下图参考，对焦花蕊区域清晰，离对焦点远的花瓣逐渐模糊 |  |
|  |  |

#### 空间感

| **元素类**元素有前后空间层次之分前景元素保持相对清晰轮廓，靠后的元素进行虚化，形成“近实远虚”的效果，使前后空间感更为明显。 |  |
| --- | --- |
| **场景类**可将元素大致分为：前景、中景、远景视觉焦点在中景保持清晰，前景和远景虚化模糊。 |  |
| **背景类**遵循“近大远小”“近实远虚”原则画面由多个弥散元素组成，远处的元素会更小，更看不清，虚化力度也会更大。（大小、虚化力度、透明度视情况叠加）​ |  |

#### 噪点质感

| 噪点颗粒一般为制作好渐变弥散图形之后，再加上杂色噪点 |  |
| --- | --- |
| 弥散粒子1.粒子颗粒，有梳有密，过渡自然；2.颗粒大小不宜过大，精致细密。​<br>​ |  |

### 常见弥散风格案例教程

#### 案例一：弥散艺术字案例（使用工具Adobe Photoshop）

通过ps滤镜“模糊画廊”工具+溶解，获得弥散效果。

**1.ps新建画布1242*1660,输入字体并将其转为智能对象**
（注：字体先不用放字太大，周边留有一些空间房子弥散效果）

**2.选中文字图层，在顶部菜单栏找到，滤镜-模糊画廊-场景模糊**​

**3.根据需要在文字不同区域，进行模糊处理**
（注：圆环处可灵活调节模糊力度调整）

**4.将混合模式改为溶解**​

**5.按快捷键 ctrl+shift+alt+E 盖印一层**​

**6.切换到通道，选择黑白对比最明显的通道，并抠出选区。**​

**7.回到图层界面，ctrl+shift+i 反选选区，ctrl+c ctrl+v 即可抠出弥散字体**​

**8.将扣除的弥散字体添加渐变颜色，并放置到合适的大小**​

##### 延展案例1

ps滤镜-模糊画廊-路径模糊

1.ps新建画布1242*1660,输入字体并将其转为智能对象，复制一个字体隐藏备存
（注：字体先不用放字太大，周边留有一些空间房子弥散效果）

**2.选中路径模糊，取消勾选居中模糊，调整合适路径，点击确认**​

**3.加上溶解之后，识别度稍微有些低**​

| 注：溶解颗粒过大，可通过等比放大画布，缩小弥散颗粒。 |  |
| --- | --- |

**3.找到之前备存的文字图层，鼠标移至缩略图处，ctrl+左键，获取选区。按快捷键shift+f6，羽化10px**​

**4.新建一个空白图层，填充选区，混合模式改成溶解**​

**5.盖印一层之后将弥散文字扣出（方法同上）**​

​

##### 延展案例2

1.新建画布1242*2208，并绘制两个圆

**2.使用蒙版工具，将圆擦除一部分**​

**3.将两个圆转为智能对象**​

**4.使用滤镜-模糊画廊-场景模糊，保持两边虚，中间实。**​

**5.给智能对象 滤镜-杂色-添加杂色，杂色数量为5**​

**6.绘制背景**​

**7.对背景进行高斯模糊，高斯模糊180**​

**8.复制一层背景，给背景添加杂色**​

**9.降低透明度为50%，并添加蒙版在不需要杂色的区域 涂抹黑色**​
