# 文章长图模版品质标准
该标准基于海报模版品质标准 ，设计时需一并参考

| **设计原则** | **设计要点** | **说明** | **正面案例** | **负面案例** |
| --- | --- | --- | --- | --- |
| **和谐统一** | 文字统一 | 选择**与主题和风格相匹配**的字体，来传达情感或强化信息 |  |  |
| 颜色 | 与**主题或行业特性相匹配**，能够更好的强化信息和情感表达 |  |  |  |
| 使用**色彩搭配工具或理论来搭配颜色**，使视觉平衡舒适，避免过于刺眼或不协调（脏乱、陈旧）的配色 |  |  |  |  |
| 排版 | 使用**版式重复原则**来保证画面的整齐统一，可以是**形式、字号、色彩、对齐方式的统一** |  |  |  |
| 背景 | 和整体**氛围调性保持统一** |  |  |  |
| **层次分明** | 文字 | 通过字**体大小（倍数增长）、颜色、加粗、下划线**等方式突出关键信息。 |  |  |
| 颜色 | 通过色彩的**明暗、冷暖**和**对比度**来区分内容的层次 |  |  |  |
| 图形/图片 | 使用**大小****、位置**来梳理图形的层次 |  |  |  |
| 通过**图像的虚实、明暗处理**，来提升画面的空间层次 |  |  |  |  |
| 排版 | 通过**箭头、线条或文字提示引导读者的视线**，让他们按照设计的顺序阅读文章内容 |  |  |  |
| 通过**亲密性和对比原则**梳理层级，**结合主题元素**（符号、色块、线条、背景）丰富画面层次，并用**点线面引导视觉动线**。 |  |  |  |  |
| **焦点强调** | 文字 | 通过**颜色、材质、字号大小、字重粗细、字效、添加图形等**方式来突出关键信息/数据 |  |  |
| 图形/图片 | 可通过**醒目的图形、大胆的图像**作为视觉焦点，作为吸引观众注意力的核心区域 |  |  |  |
| 排版 | 使用**强烈的对比配色（颜色对比/面积对比）**技法，来突出关键要传达的信息或视觉元素 |  |  |  |
| **传达清晰****标标题≥36px，正文≥24px，注释≥18px题≥36px，正文≥24px，注释≥18p** | 文字清晰易读（）（（）免超过3种字体。避免超过3种字体。 | 选择**清晰易读的字体**，避免使用过于花哨的字体影响阅读体验。​ *字体数量**不宜太多**，一般是2-3种 |  |  |
| **分块设计：**将长图划分为多个模块，每个模块对应一个核心观点或信息点，增强可读性 | **规划内容结构**：合理划分长图的内容板块，可使用小标题、分隔线等区分不同部分，使内容层次分明。 |  |  |  |
| **视觉连贯性**用重复元素串联模块：如每个模块左上角放置相同风格的序号图标； |  |  |  |  |
| **模块节奏****图文比例**：文主导型：文字占比70%，配小图图主导型：大图占60%，文字精简 |  |  |  |  |
|  |  |  |  |  |
| **分割方式多样化**硬分割：色块、线框、几何图形；软分割：留白、渐变过渡、半透明蒙版叠加；动态分割：斜切、波浪线、穿插式插图*模块间留白≥1.5倍行高，避免视觉粘连；*模块上下错开排列，形成Z字动线；模模块上下错开排列，形成Z字动线；块上下错开排列，形成Z字动线；； |  |  |  |  |
| 图形/图片 | **挑选优质素材**：使用高清、高质量的图片、图标和插图等素材，避免使用模糊、低分辨率或版权不明的素材。 |  |  |  |
| 图片和文字内容应**紧密结合**，图片/图标应直观地辅助文字说明，增强信息的**传达效果**。 |  |  |  |  |
| 背景 | 确保**前景文字与背景有足够的对比度**，内容展示清晰 |  |  |  |
| **简洁明了****** | 文字 | 色彩搭配：选择**简洁、协调**的色彩方案，通常以**1-2种主色调搭配1-2种辅助色。** |  |  |
| **避免信息过载**：每个模块或页面展示的信息量应适中，避免一次性展示过多信息。 |  |  |  |  |
| 图形 | **避免过度装饰和复杂的设计布局。** |  |  |  |
| 背景 | 避免使用**复杂的图案/醒目**的背景图案，以免分散对主要内容的注意力 |  |  |  |
| **留白得当** | 文字 | 设置**合适的字距**，避免文字过于拥挤或稀疏，提高阅读舒适度。 |  |  |
| 图形/图片 | 涉及图形和图像结合，需**注意保留空间**，避免图像/文字被**遮挡、挤压** |  |  |  |
| 排版 | 在长图中保持空间的平衡，**避免某些部分过于空旷或拥挤。** |  |  |  |
| 需考虑需求调性，利用**版面率知识，创造出相匹配的画面气质**低低版面率-留白多，画面气质更典雅、安静高版面率-留白少，画面气质更活泼、热闹） |  |  |  |  |
|  |  |  |  |  |
| **细节丰富** | 图形/图片 | 使用与内容的紧密相关的**图表来辅助文字内容**，增强信息的直观性和易理解性，同时也能增加设计的美观性数据模块优先信息图表 |  |  |
| 在长图中加入简单的小元素效果，可以大大增加画面趣味性（如右侧：鼠标箭头、皇冠等） |  |  |  |  |
| **创意创新****** | 文字 | **创意字体：**适当使用创意字体来突出品牌或者主题，但需注意不要过度使用，以免影响阅读体验 |  |  |
| 图形/图片 | **创意图形：**降低理解成本，增强记忆点，适当使用可以强化品牌符号 |  |  |  |
