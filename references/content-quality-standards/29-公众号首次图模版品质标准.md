# 公众号首次图模版品质标准
该标准基于 ，设计时需一并参考

| **设计原则** | **设计要点** | **说明** | **正面案例** | **负面案例** |
| --- | --- | --- | --- | --- |
| **和谐统一** | **契合用途调性** | 首图和次图的设计风格应**与公众号的用途高度一致**。如时尚类公众号，图片风格可时尚、精致且富有创意；知识科普类公众号，图片可能更倾向简洁明了、逻辑清晰，运用与知识内容相关的元素和配色，传递专业、可靠的形象。 |  |  |
| **保持系列感** | 首图和次图之间，以及不同文章的图片之间应**保持一定的系列感**。如使用相似的色彩体系、字体风格、图形元素等，让读者在浏览公众号时能形成强烈的视觉印象，增强辨识度。 |  |  |  |
| **色彩搭配合理** | 可以根据文章主题和情感选择色彩如欢快的文章用明亮活泼的色彩，严肃的文章用沉稳色调。 |  |  |  |
| **图文契合** | 图像内容要与文章主题紧密相关，能引发读者兴趣。 |  |  |  |
| **强调焦点** | **主题突出** | **首图**要能**迅速传达文章核心主题**，让读者一眼明白文章大致内容。可通过图片主体、文字标题等方式突出。**次图**虽信息承载量相对少，但也应**与文章主题有所关联**，辅助强化主题印象。如一篇美食推荐文章，首图可直接展示美食图片，并搭配醒目的美食名称和 “推荐” 字样；次图可展示美食的局部特写或制作场景。 |  |  |
| **主体图形突出** | 对于首图，图像**主体应突出**，**构图简洁有力**；次图可选择与首图相关联的局部图或补充图，增加视觉层次。 |  |  |  |
| **简洁明了** | **图片简洁** | **避免图片信息过于繁杂**，干扰主题表达。去除不必要的装饰和元素，确保关键信息突出。**背景一致时接缝统一** |  |  |
| **文字简洁** | 控制字数，**避免长篇大论**。 |  |  |  |
| **传达清晰** | **文字易读** | 选择**清晰易读的字体**，避免使用过于花哨、难以辨认的字体。根据图片整体风格和排版，合理**调整字体大小、颜色和字间距**，确保文字在不同设备上都能轻松阅读。 |  |  |
| **色彩对比清晰** | **注意色彩对比度**，保证文字和图片内容清晰可辨。 |  |  |  |
| **图片清晰** | 考虑读者阅读公众号文章的场景多样，图片设计要在各种场景下都能清晰展示。如在光线较暗的环境下，色彩和文字依然可辨；在不同屏幕分辨率下，图片质量不受影响。 |  |  |  |
| **创意创新** | **色彩吸引** | 选择**鲜明、协调的色彩组合**吸引读者目光。 |  |  |
| **话题吸引** | 选用**高话题度的文案。** |  |  |  |
| **图片独特** | 选用具有**视觉冲击力**的图像、元素运用**独特的创意和设计手法**，使图片脱颖而出。如采用独特的排版方式、创意合成、巧妙的比喻或象征等，让读者产生好奇心，进而点击阅读。 |  |  |  |
| **易于延展** | **背景拆分** | 考虑背景的尺寸延展，首次图背景需进行拆分，矢量背景需保持颜色可编辑，无明显分界的背景拆分后背景需填充一致的颜色；​<br>​ |  |  |
| 若首次图接缝处有具体元素，需保持元素完整，可通过背景拆分进行元素遮挡； |  |  |  |  |
| **元素易用** | **保持完整和独立****可以成为独立个体使用**的元素，需要保持独立图层，主要是方便用户对画面进行一些元素上的移动和调整，保留模板的可编辑性最大化;相隔太远的元素，一定要分割开成为独立的图层，包括背景上的装饰元素，否则也不方便进行元素的移动和调整；元素可以超出画布注意：C4D相关作品也需要遵循元素完整和独立原则，需要分层渲染，尽量做到各元素可单独移动。​<br>**单个元素裁切干净**注意元素裁切干净，不要**有多余的元素在看不见的地方**，会导致图层过大，无法导入的问题​<br>**矢量图形：**但凡涉及到矢量图形（基础线条/色块/图形/图框/icon等）都需保留矢量可编辑，便于用户修改颜色。**编辑器内需检查是否能修改颜色** | →→ |  |  |
| **文本易用** | **文字背景特效还原：**文字+纯色底框色块的设计样式，需在编辑器内使用文字背景功能直接还原。说明：除个性化风格外，需尽量设置还原 |  |  |  |
