# 特效素材品质标准
### 设计原则及标准

| 设计原则 | 标准说明 | 正面案例 | 反面案例 |
| --- | --- | --- | --- |
| 元素完整 | 确保主体元素完整，没有缺失的部分。构图合理裁剪，不影响素材应用。 |  |  |
| 线条流畅 | 线条应该流畅自然，避免出现生硬或断裂的线条。 |  |  |
| 色彩协调 | 色彩搭配和谐，遵循色彩理论，以确保视觉效果的和谐统一。 |  |  |
| 质感真实 | 表现特效元素对应材质真实质感，具体不同材质表现可参照下方设计要点说明表格。 |  |  |
| 有动态感 | 特效素材应该能够表现出动态变化，即使是静态图像也应该能够暗示出动态过程。 |  |  |
| 符合物理规律 | 特效素材的设计应该遵循现实世界的物理规律，如火焰的上升、烟雾的扩散、光线的反射、水珠的流动性等。 |  |  |
| 易于合成 | 合理的透明度与渐变效果，在不同背景上都能很好展示和应用不需要复杂的调整就能与现有其他视觉元素和谐融合。 |  |  |
| 画质清晰 | 保证高清画质，100%放大后图形边缘清晰锐化，无模糊感，无锯齿马赛克，无杂色像素。 |  |  |

### 不同材质特效的设计要点说明

| 材质特效 | 说明 | 正面案例 | 反面案例 |
| --- | --- | --- | --- |
| 水滴/水珠 | 通过细致的高光阴影及透明度来体现水滴体积感和透明液体质感；形状圆润，边缘平滑，造型优美自然；水滴为低落状态的水珠，受重力影响呈现一头圆形一头尖状的水滴形状；水珠为漂浮或附着状态，形状为圆形或以圆形为基础的变形；多个水珠形状需有变化，但光影保持一致；颜色为灰白色或者淡蓝色，表现水珠洁净感； |  |  |
| 水波纹 | 表现水面波纹的真实感，真实感的水波纹应该能够模拟水面的波动、反射和折射效果；有明显的层次感，波纹强弱对比，模拟水波纹动态效果；颜色为灰白色或者淡蓝色，表现水面洁净感； |  |  |
| 水花 | 模拟真实水花飞溅效果，高速快门拍摄状态下的静态画面；强烈的动态感，能够表现出水花运动轨迹和方向；遵从物理规律的同时，让水花构图和造型更具创意和艺术性；颜色为灰白色或者淡蓝色，表现水花洁净感； |  |  |
| 雨水 | 模拟真实下雨效果，雨滴呈现动态模糊拖尾，分布均匀随机；雨滴有前后层次感，有不同大小和透明度对比，表现出空间感；雨滴方向大体一致，尽量避免角度交叉，画面凌乱的感觉；如果有地面积水，需表现出雨滴入水后的水花和涟漪；如有玻璃，表现雨水打到玻璃上的附着和滴落流淌效果； |  |  |
| 雪花 | 模拟真实下雪效果，雪花大小和速度应该是不一致的，分布随机；雪花有前后层次感，有不同大小和透明度模糊度对比，表现出空间感；通过动态模糊表现风吹动雪花的效果，运动方向大体一致，角度可交叉；写实风格雪花形状以圆点斑点表现，插画风格可用显微雪花结构表现； |  |  |
| 冰块 | 模拟真实冰块的质感，添加表面的凹凸、内部的气泡等增强真实感；通过光的反射和折射表现冰块固态效果和透视效果；冰块造型以立方体为主，通常会有一定的挤压或融化变形；颜色为灰白色或者淡蓝色，表现冰块剔透清凉的视觉感受； |  |  |
| 气泡 | 需有良好的透明质感，营造出轻盈通透的效果；模拟光线在气泡表面的反射和折射效果，增强立体感和真实感；造型充盈圆润，表现气泡上升漂浮感；多个气泡错落分布，表现出空间感，表面光线效果保持统一；颜色以多彩或清冷单色为主，也可以纯白呈现干净的视觉感受； |  |  |
| 泡沫 | 能清晰分辨出泡沫表面的细小气泡，表现泡沫绵密的特点；通过整体的高光和阴影，表现泡沫体积感；通过不同的气泡疏密和大小分布，表现泡沫的厚度和密度变化；颜色以纯白为主，表现泡沫洁净感； |  |  |
| 透气 | 使用线条和模糊效果表现气流走势和气体形状，将“透气”概念可视化；线条要流畅，看的出气流走势；线条分布均匀，不相互交叉打架，利用透明度和颜色区分主次关系；颜色以白色或蓝色为主，表现气流的轻盈感； |  |  |
| 火焰 | 仅表现火焰，不表现可燃物；火焰造型要逼真，线条流畅有动态感，表现火焰上升蔓延和摇曳的动感；颜色以黄红为主，越靠近焰心的部分，温度越高，颜色越接近白色；增加漂浮的粒子和火舌，提升火焰真实感； |  |  |
| 烟雾 | 表现烟雾的自然特性，流动、扩散、渐变消失等，保证效果真实性；能看得出烟雾的走势和扩散方向，表现出动态感；通过透明度变化，表现雾气朦胧的感觉；颜色以白色或灰色为主，表现烟雾的轻盈感； |  |  |
| 云朵 | 不规则的外形，使用柔和的边缘和曲线来模拟云朵的蓬松感；云朵层次要体现，通过前后云层堆叠来营造层次感；需要有明暗对比，模拟自然光线照射在云朵上的效果，表现云朵的立体感；使用噪点、笔刷或滤镜来模拟云朵的纤维状结构，同时保持云朵轻盈感；颜色以白色或灰白色为主，暗面可根据天空颜色或光照条件进行色彩调整； |  |  |
| 光线 | 光线的线条需要流畅，不能有生硬断裂或不和谐的弯曲抖动；发光效果明显，强调光源或高光部分，最亮的部分颜色最实，最接近白色，距离高光或光源越远，光效和颜色越淡，呈现光的衰减和散射效果；色彩以单色为主，颜色一般以白色、黄色或蓝色为主； |  |  |
| 光圈 | 由一条或多条光线组成的圆形或椭圆形光圈造型，光圈的线条要求流畅，符合圆形或椭圆形的曲率；如有透视角度，需绘制准确，近大远小，同时光线自身的亮度和粗细也应有规律地变化；发光效果明显，强调光源或高光部分，最亮的部分颜色最实，最接近白色，距离高光或光源越远，光效和颜色越淡，呈现光的衰减和散射效果；色彩以单色为主，颜色一般以白色、黄色或蓝色为主； |  |  |
| 光影 | 模拟光透过窗户或者植物缝隙，投到墙上的光影效果；窗户结构不宜太过复杂，以田字形结构为主；植物影子要有斑驳感，边缘模糊，造型优美不凌乱；越靠近光源的造型边缘越清晰，越远离光源的造型边缘越模糊；颜色以透明白色或灰色为主，白色绘制光亮，灰色绘制投影，光影素材单独绘制光亮或投影，二选一，不同时绘制； |  |  |
| 光束 | 表现光源放射出光束，光为锥状或放射状，由多条光线或多个光面组成；光的强度和边缘清晰度，离光源越近越清晰，离光源越远越模糊，直至渐变消失；光越强的位置，颜色越实，越接近白色；色彩以单色为主，颜色一般以白色、黄色或蓝色为主； |  |  |
| 光晕/光斑 | 模拟透过镜头观察光源，呈现的光晕和虚化光斑效果；光晕为圆圈形状，它的圆心与光源应在同一直线上；需表现光晕的晕染效果；光斑需表现出层次感，不同大小、虚实、透明度的光斑随机错落分布；彩色光斑颜色渐变需符合光谱颜色顺序，即彩虹色顺序； |  |  |
| 撕纸 | 表现纸张撕边的自然造型和纸张纤维感肌理；撕边的造型要流畅自然，随意的不规则边缘，呈现真实感；模拟光线对撕纸边缘的影响，添加阴影和高光，使撕纸效果更加立体和真实；纸张撕开边缘的颜色以白色灰色为主，更具适用性； |  |  |
| 闪电 | 模拟自然界闪电的形态，呈现树根状结构，首尾清晰，造型完整；闪电需有外发光效果，增强闪电的光亮感；色彩单色，一般以白色、蓝色、紫色、黄色为主； |  |  |
| 爆炸 | 呈现爆炸火焰放射状喷射，发光，充满能量的感觉；细节丰富，火焰层次明显，伴随爆炸喷射的燃烧粒子、烟雾，增加真实感；粉尘爆炸，不带火焰，仅以粒子爆炸效果表现，不发光，突出体积感和动态感； |  |  |
| 破碎 | 模拟玻璃或墙面破碎效果，有明显的孔洞或飘浮的碎屑；裂缝线条以直线为主，以破洞为中心成放射状，增加不规则的线段表现破碎真实感；需要表现玻璃或者墙面的厚度，增加立体感；需要表现玻璃的透明质感； |  |  |
